# 전체 로드맵: 유튜브 채널 자동화 프로젝트

> 최종 수정: 2026-03-03

---

## 목표

키워드/주제 입력만으로 영상 제작 → 업로드까지 자동 완료되는 파이프라인 구축.
4개 채널을 단계적으로 자동화하여 **최소 시간 투입으로 패시브 수익 + 브랜딩** 달성.

---

## 채널 로드맵 요약

| 순서 | 채널 | 자동화 | 목표 | 착수 시점 |
|------|------|--------|------|----------|
| 1 | 힐링 음악 (기존) | 전체 | 패시브 수익 + 자동화 연습 | **즉시** |
| 2 | AI 뉴스 쇼츠 | 전체 | 수익 + AI 자동화 스킬 습득 | Phase 1 완료 후 |
| 3 | 창업 도전기 | 반자동 | 브랜딩 + 홍보 + 장기 자산 | Phase 2 완료 후 |
| 4 | AI 활용법 강좌 | 반자동 | 강좌 수익화 | 여유 시 확장 |

---

## Phase 1: 힐링 음악 자동화

**기간:** 2~3주
**목표:** 키워드 입력 → 3시간 롱폼 영상 생성 → 업로드 원스텝 파이프라인

### 파이프라인 흐름

```
키워드 + 프리셋 입력 (예: "cozy winter night" + fireplace)
  → Gemini API로 AI 이미지 생성
  → 켄번스 효과 + 루프 영상 오버레이 (장작불, 눈, 비 등)
  → 자연 소리 합성 (Freesound API)
  → 3시간 롱폼 확장 + 썸네일 생성
  → YouTube Data API로 업로드
```

### 세부 마일스톤

| # | 작업 | 산출물 | 상태 |
|---|------|--------|------|
| 1-1 | Gemini API 이미지 생성 모듈 | `image_generator.py` | 미착수 |
| 1-2 | 루프 오버레이 소스 수집 | `assets/overlays/` | 미착수 |
| 1-3 | 켄번스 + 오버레이 영상 합성 | `video_composer.py` | 미착수 |
| 1-4 | Freesound API 소리 확보 모듈 | `audio_source.py` | 미착수 |

| 1-6 | 썸네일 자동 생성 | `thumbnail_gen.py` | 미착수 |

## Phase 2: AI 뉴스 쇼츠 자동화

**기간:** 2~3주 (Phase 1 완료 후)
**목표:** RSS 뉴스 수집 → 1분 쇼츠 영상 자동 생성 → 업로드

### 파이프라인 흐름

```
RSS 피드 크롤링 (TechCrunch, The Verge 등)
  → Claude API로 한국어 요약 + 쇼츠 대본 생성
  → TTS API로 음성 합성
  → 뉴스 관련 이미지 수집 + 자막 생성
  → FFmpeg/Remotion으로 영상 조합
  → YouTube Data API로 업로드
```

### 세부 마일스톤

| # | 작업 | 산출물 | 상태 |
|---|------|--------|------|
| 2-1 | RSS 뉴스 수집 모듈 | `ai-news-shorts/src/news_collector.py` | 미착수 |
| 2-2 | Claude API 스크립트 생성 | `ai-news-shorts/src/script_gen.py` | 미착수 |
| 2-3 | TTS 음성 합성 모듈 | `ai-news-shorts/src/tts_engine.py` | 미착수 |
| 2-4 | 이미지 수집 + 자막 생성 | `ai-news-shorts/src/visual_gen.py` | 미착수 |
| 2-5 | 쇼츠 영상 조합 | Phase 1의 FFmpeg 유틸 재활용 | 미착수 |
| 2-6 | 통합 파이프라인 | `ai-news-shorts/src/pipeline.py` | 미착수 |
| 2-7 | 테스트 업로드 + 검증 | 실제 업로드 확인 | 미착수 |

## Phase 3: 창업 도전기 반자동화

**기간:** 필요 시 점진적 진행
**목표:** 내용 입력 → 설명 영상 자동 제작 (촬영 영상은 수동)

### 자동화 범위

| 구분 | 방식 |
|------|------|
| 설명 영상 (창업 과정, AI 활용기) | 내용 입력 → 스크립트 → TTS → 영상 자동 제작 |
| 일상/근황 영상 | 폰 촬영 + 최소 편집 (수동) |

---

## Phase 4: AI 활용법 강좌 (미래)

- 창업 도전기에서 반응 좋은 AI 활용 주제를 체계적 강좌로 확장
- Phase 1~3 자동화 파이프라인 자체를 "어떻게 만들었는지" 강좌화 가능
