# Phase 1: 힐링 음악 자동화 설계

> 최종 수정: 2026-03-03

---

## 개요

키워드 입력만으로 **AI 이미지 생성 → 부분 움직임 영상 변환 → 자연 소리 합성 → 3시간 롱폼 영상 → 업로드**까지 자동 완료되는 파이프라인.

---

## 파이프라인 전체 흐름

```
[입력] 키워드 + 프리셋 선택 (예: "cozy fireplace, winter night" + fireplace 프리셋)
  │
  ├─ Step 1. 이미지 생성 (Gemini API - Nano Banana 2)
  │    → 프롬프트 템플릿 + 키워드 조합 → 고품질 이미지 생성
  │
  ├─ Step 2. 영상 변환
  │    → 정적 이미지 + 루프 영상 오버레이 (장작불, 눈, 비 등)
  │    → FFmpeg로 자동 합성 (화면 효과 최소화 — 배경용 영상 특성 반영)
  │
  ├─ Step 3. 자연 소리 합성
  │    → Freesound API로 소리 소스 확보 (장작불, 풀벌레, 빗소리 등)
  │    → FFmpeg로 영상 + 소리 합성 + 페이드인/아웃
  │
  ├─ Step 4. 롱폼 확장
  │    → 영상 + 소리를 3시간으로 루프 확장
  │
  ├─ Step 5. 썸네일 생성
  │    → Step 1 이미지에 텍스트 오버레이 (Pillow)
  │
  └─ Step 6. YouTube 업로드
       → YouTube Data API v3 (제목/설명/태그/썸네일 포함)
```

---

## Step별 상세 설계

### Step 1. 이미지 생성

| 항목 | 내용 |
|------|------|
| API | Gemini API (Nano Banana 2 / gemini-2.5-flash-image) |
| 비용 | 무료 티어 하루 500장 / Gemini Pro 구독 시 추가 혜택 |
| 방식 | 프롬프트 템플릿 + 키워드 조합 |
| 출력 | 1024x1024 이상 고해상도 이미지 |

**프롬프트 템플릿 예시:**

```
카테고리: fireplace
프롬프트: "A cozy modern living room with a large stone fireplace centered in the frame,
          warm lighting, {keyword}, photorealistic, 4K, ambient atmosphere"
```

**카테고리별 템플릿:**

| 카테고리 | 키워드 예시 | 비고 |
|----------|-----------|------|
| fireplace | winter night, christmas, rustic cabin | 벽난로 위치 하단 중앙 고정 |
| rain | city window, forest, ocean cliff | 창문/전체 화면 |
| forest | summer, autumn leaves, misty morning | 전체 화면 |
| snow | mountain cabin, village, park bench | 전체 화면 |
| ocean | sunset beach, moonlit waves, tropical | 전체 화면 |

### Step 2. 영상 변환

정적 이미지를 베이스로 하고, 루프 영상 오버레이만 추가하여 최소한의 움직임을 부여.
힐링 영상은 배경으로 켜놓는 용도이므로 **화면 효과를 최소화**하는 것이 핵심.

#### 루프 영상 오버레이

| 오버레이 유형 | 적용 방식 | 위치 |
|-------------|----------|------|
| 장작불 | 크로마키/알파 루프 영상 | 하단 중앙 (벽난로 프리셋) |
| 눈 내리는 효과 | 반투명 루프 영상 | 전체 화면 |
| 빗방울 | 반투명 루프 영상 | 전체 화면 |
| 반딧불 | 반투명 루프 영상 | 전체 화면 |
| 파도/물결 | 하단 오버레이 | 하단 1/3 |

**오버레이 소스 확보:**
- Pixabay/Pexels에서 투명 배경 또는 어두운 배경 루프 영상 수집
- 10~30초 루프 영상 → 카테고리별로 `healing-music/assets/overlays/`에 저장
- 한번 확보하면 재사용 가능

**프리셋 구조 (JSON):**

```json
{
  "fireplace": {
    "prompt_template": "A cozy room with a large fireplace centered...",
    "overlay": {
      "source": "overlays/fireplace_loop.mp4",
      "position": "center_bottom",
      "size": "30%",
      "blend": "screen"
    },
    "audio": {
      "keywords": ["fireplace crackling", "wood burning"],
      "fallback": "audio/fireplace_default.mp3"
    },
    "thumbnail": {
      "title": "Cozy Fireplace Sounds",
      "duration_text": "3Hours",
      "font": "GreatVibes-Regular.ttf",
      "text_color": "#FFFFFF",
      "shadow": true
    }
  },
  "rain": {
    "prompt_template": "A window view with rain outside...",
    "overlay": {
      "source": "overlays/rain_loop.mp4",
      "position": "full_screen",
      "blend": "screen",
      "opacity": 0.4
    },
    "audio": {
      "keywords": ["rain ambient", "thunder distant"],
      "fallback": "audio/rain_default.mp3"
    }
  }
}
```

### Step 3. 자연 소리 합성

| 항목 | 내용 |
|------|------|
| 소스 | Freesound API (무료, CC 라이선스) |
| 검색 | 프리셋의 audio.keywords로 자동 검색 + 다운로드 |
| 대안 | 프리셋별 기본 소리 파일 (fallback) |
| 처리 | FFmpeg로 페이드인/아웃 + 볼륨 조절 + 영상 합성 |

### Step 4. 롱폼 확장

| 항목 | 내용 |
|------|------|
| 목표 길이 | 1~3시간 (설정 가능) |
| 방식 | 영상 + 소리를 FFmpeg `stream_loop`로 반복 |
| 전환 효과 | 이미지 2~3장 생성 → 각각 켄번스 적용 → 크로스페이드로 연결 |

### Step 5. 썸네일 생성

기존 채널 썸네일 스타일을 그대로 재현하는 자동 생성:

| 항목 | 내용 |
|------|------|
| 베이스 | Step 1에서 생성한 이미지 (1280x720 리사이즈) |
| 처리 | Pillow로 텍스트 + 로고 + 시간 표시 오버레이 |

**썸네일 레이아웃 (기존 채널 기준):**

```
┌──────────────────────────────────┐
│ [로고]                  3Hours   │  ← 좌상단: 채널 로고 아이콘
│                                  │  ← 우상단: 시간 표시 (흰색)
│                                  │
│        Relaxing Forest Sound     │  ← 중앙: 제목 텍스트
│                                  │     필기체(cursive) 폰트
│                                  │     흰색 + 드롭쉐도우
│                                  │
└──────────────────────────────────┘
         [AI 생성 배경 이미지]
```

**텍스트 스타일:**

| 요소 | 스타일 |
|------|--------|
| 제목 | 필기체 폰트 (예: Great Vibes, Dancing Script), 흰색, 중앙 배치, 드롭쉐도우 |
| 시간 | 산세리프 폰트, 흰색, 우측 상단, 작은 크기 |
| 로고 | 채널 아이콘 PNG, 좌측 상단, 약 50x50px |

**프리셋별 제목 템플릿:**

| 카테고리 | 제목 예시 |
|----------|----------|
| fireplace | "Cozy Fireplace Sounds", "Crackling Fire ASMR" |
| rain | "Relaxing Rain Sounds", "Rainy Night Ambience" |
| forest | "Relaxing Forest Sound", "Peaceful Nature ASMR" |
| snow | "Snowy Night Ambience", "Winter Wonderland" |
| ocean | "Ocean Wave Sounds", "Healing Sea Waves" |

### Step 6. YouTube 업로드

| 항목 | 내용 |
|------|------|
| API | YouTube Data API v3 |
| 인증 | OAuth 2.0 (최초 1회 수동 인증 → 토큰 저장) |
| 메타데이터 | 프리셋별 제목/설명/태그 템플릿 자동 생성 |
| 스케줄링 | 업로드 시 공개 예약 시간 설정 가능 |

## 프로젝트 구조

```
healing-music/
├── src/
│   ├── pipeline.py              # 메인 파이프라인 (원스텝 실행)
│   ├── image_generator.py       # Gemini API 이미지 생성
│   ├── video_composer.py        # 켄번스 + 오버레이 합성
│   ├── audio_source.py          # Freesound API 소리 확보
│   ├── longform_builder.py      # 롱폼 확장 (루프 + 크로스페이드)
│   └── thumbnail_gen.py         # 썸네일 생성
├── config/
│   ├── presets.json             # 카테고리별 프리셋 정의
│   ├── prompts.json             # 이미지 프롬프트 템플릿
│   └── .env                     # API 키 (git 제외)
├── assets/
│   ├── overlays/                # 루프 영상 소스 (장작불, 비, 눈 등)
│   ├── audio/                   # 기본 자연 소리 파일 (fallback)
│   └── fonts/                   # 썸네일용 폰트
├── output/                      # 생성된 영상 (git 제외)
└── README.md
```

## 리스크 및 대응

| 리스크 | 대응 |
|--------|------|
| Gemini API 무료 한도 초과 | 하루 1~2장이면 충분, 한도 여유 |
| Freesound 라이선스 주의 | CC0/CC-BY만 필터링, 크레딧 자동 삽입 |
| 오버레이 위치 부정확 | 이미지 프롬프트에서 벽난로 등 위치 고정 + 프리셋별 좌표 정의 |
| FFmpeg 인코딩 시간 | 3시간 영상 인코딩 오래 걸림 → GPU 가속 또는 해상도 조절 |
| YouTube API 할당량 | 일일 업로드 제한 고려, 스케줄링으로 분산 |
