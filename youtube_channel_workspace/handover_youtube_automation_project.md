## 2. 채널 운영 계획 (총 4개 채널)

### 채널별 개요

| 채널 | 콘텐츠 | 자동화 수준 | 핵심 목적 |
|------|--------|------------|----------|
| 힐링 음악 (기존) | 힐링 음악 + 영상 1~3시간 롱폼 | 전체 자동화 | 패시브 수익 + 자동화 연습 |
| AI 뉴스 쇼츠 (신규) | AI 관련 뉴스 요약 쇼츠 (1분 이내) | 전체 자동화 | 수익 + AI 자동화 스킬 습득 |
| 창업 도전기 (신규) | 창업 준비 과정, AI 활용기, 일상/근황 | 반자동화 | 브랜딩 + 홍보 + 소통 + 장기 자산 |
| AI 활용법 강좌 (미래) | AI 도구 활용법 체계적 강좌 | 반자동화 | 창업 도전기에서 반응 좋은 주제 확장 |

### 채널별 상세

#### 2-1. 힐링 음악 (전체 자동화)

| 단계 | 방법 |
|------|------|
| 음악 | YouTube Audio Library API 자동 다운로드 또는 AI 음악 생성 (Suno/Udio, 저작권 주의) |
| 영상 소스 | Pexels/Pixabay API로 키워드 기반 자동 다운로드 (fireplace, rain, forest 등) |
| 편집 | FFmpeg 스크립트로 영상 반복 + 음악 합성 + 페이드인/아웃 |
| 썸네일 | Leonardo.ai 또는 Canva API로 템플릿 기반 자동 생성 |
| 업로드 | YouTube Data API v3로 제목/설명/태그 포함 자동 업로드 |

- **목표:** "키워드 입력 -> 영상 완성 -> 업로드" 원스텝 파이프라인

#### 2-2. AI 뉴스 쇼츠 (전체 자동화)

| 단계 | 방법 |
|------|------|
| 뉴스 수집 | RSS 피드 크롤링 (TechCrunch, The Verge, AI 관련 블로그) |
| 요약/스크립트 | Claude API로 한국어 요약 + 쇼츠 대본 생성 |
| 음성 | TTS API (ElevenLabs, Google TTS, 또는 Edge TTS) |
| 영상 | 뉴스 관련 이미지 자동 수집 + 자막 생성 후 FFmpeg/Remotion으로 조합 |
| 업로드 | YouTube Data API로 자동 업로드 + 스케줄링 |

- **참고:** 쇼츠 RPM은 롱폼 대비 1/10~1/50로 낮음. 수익보다는 자동화 스킬 습득이 주목적
- **수익화 조건:** 구독자 1,000명 + 쇼츠 조회 1,000만회

#### 2-3. 창업 도전기 (반자동화 + 수동)

**자동화 가능 콘텐츠:**
- 창업 관련 과정 설명 영상 (내용 입력 -> 스크립트 자동 생성 -> 영상 제작)
- AI 활용기 영상 (같은 파이프라인)

**수동 콘텐츠:**
- 일상/근황: 폰으로 5~10분 가볍게 촬영 + 최소 편집

**콘텐츠 카테고리:**

| 카테고리 | 예시 |
|----------|------|
| 창업 준비기 | 사업계획서 작성기, 특허 출원 과정, 프로젝트 후기 |
| 대학원 일상 | 연구실 루틴, 논문 스트레스, 실험 실패담 |
| AI 활용기 | 펩타이드 파이프라인 개발 과정, Claude/ChatGPT 활용법 |
| 근황/회고 | 직장에서 대학원 전환 이야기, 진솔한 고민 |

- **핵심 가치:** 장기 브랜딩 자산 + 창업 후 제품 홍보 채널 + 심리적 해소

#### 2-4. AI 활용법 강좌 (미래 확장)
- 창업 도전기에서 AI 활용기 영상 중 반응 좋은 주제를 체계적 강좌로 확장
- 시기: 여유 생기면 진행

### 채널 간 시너지
- 창업 도전기의 AI 활용기 -> AI 강좌 채널 소스
- AI 뉴스 쇼츠에서 다룬 기술 -> 강좌 실습으로 연결
- 자동화 파이프라인 자체가 -> "이걸 어떻게 만들었는지" 강좌 콘텐츠
- **하나를 만들면 여러 채널에 재활용 가능한 구조**
