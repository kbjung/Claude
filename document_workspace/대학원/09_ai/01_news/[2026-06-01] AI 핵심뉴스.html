Claude Opus 4.8 — Dynamic Workflows 탑재

Anthropic 5월 28일 에이전틱
Anthropic이 Claude Opus 4.8을 출시했다. 전작 대비 41일 만의 신규 모델로, 코딩 에이전틱 벤치마크 69.2%(+4.9%p), 지식 작업 점수 1890(+137) 기록. Claude Code에 대규모 작업을 자율 분해·처리하는 Dynamic Workflows 기능이 추가됐다.
  • 사용자가 태스크 노력 수준 직접 조정 가능
  • 패스트 모드(2.5× 속도) 이전 대비 3배 저렴
  • 진행 상황에 대한 불확실성 적극 공지 — 정직성 강화
  • 가격 동결: 입력 $5 / 출력 $25 per 1M 토큰

GPT-5.5 Instant — ChatGPT 기본 모델 교체

OpenAI 5월 5일 개인화
OpenAI가 GPT-5.5 Instant를 모든 ChatGPT 사용자 기본 모델로 배포했다. 고위험 프롬프트(의학·법률·금융) 환각이 GPT-5.3 대비 52.5% 감소. Gmail·과거 대화·업로드 파일을 자동 참조해 개인화된 답변을 제공한다.
  • 코드 작성·디버깅·데이터 분석·소프트웨어 조작 강화
  • 불필요한 이모지 제거, 더 직접적인 응답 톤
  • GPT-4.5는 6월 27일 ChatGPT에서 완전 종료 예정
  • API: GPT-5.5 Pro는 4월 24일부터 이미 사용 가능

DeepSeek V4 — 오픈웨이트 1T+ 파라미터 모델

DeepSeek 4월 24일 프리뷰 오픈소스
중국 DeepSeek이 DeepSeek-V4-Pro(1.6T 파라미터, 49B 활성화)와 V4-Flash(284B)를 공개했다. 1M 토큰 컨텍스트, CSA+HCA 하이브리드 어텐션으로 단일 토큰 추론 FLOPs 27% 수준 달성. GPT-5.5·Opus 4.7급 에이전틱 벤치마크를 오픈웨이트·저비용 API로 제공한다.
  • 자체 호스팅 가능 — 클로즈드 모델 의존도 탈피
  • KV 캐시 사용량 10% 수준으로 장문 추론 비용 절감
  • 서방 프론티어 모델 대비 API 비용 크게 낮음

EU AI Act — 8월 2일 GPAI 집행 권한 발효

EU AI Act GPAI
2026년 8월 2일부터 EU 집행위원회가 GPAI 모델 공급자에 대한 문서 요청·평가·조치 명령·과징금 부과 권한을 발동할 수 있다. 동일자로 투명성 의무(제50조) — AI 생성 콘텐츠 레이블링 — 도 적용된다.

  • 2025년 8월 이전 출시 GPAI 모델 준수 기한: 2027년 8월 2일
