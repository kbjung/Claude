RFdiffusion3 — 원자 수준 단백질 설계 공개 출시

노벨화학상 수상자 David Baker 교수팀(워싱턴대)이 RFdiffusion3를 오픈소스(MIT 라이선스)로 공개했다. 기존 버전 대비 10배 빠른 추론 속도를 갖추며, 단백질·DNA·소분자 등 세포 내 모든 분자 유형과 상호작용하는 단백질을 설계할 수 있다.
  • 핵심 혁신 — 잔기당 14원자 전체-원자 표현(all-atom representation)으로 리간드 결합을 scratch부터 생성
  • 설계 파이프라인 — RFdiffusion3 → ProteinMPNN/LigandMPNN → AlphaFold3 검증
  • 지원 기능 — 수소결합 원자 지정, 용매 접근성 조절, DNA 공동 디퓨전
  • 학습 데이터 — PDB 구조체(~2024.12) + AlphaFold2 예측 구조 포함

LigandForge — 150,000개 펩타이드 후보를 3분 만에 생성

Ligandal社의 웹 기반 펩타이드 설계 플랫폼 LigandForge가 출시됐다. 단일 패스 이산 디퓨전(single-pass discrete diffusion)과 DeltaForge 열역학적 스코어링(r=0.83)으로 초고속 후보 생성이 가능하다.
항목 내용
속도 초당 1,000개+ 펩타이드 바인더 생성 (기존 방법 대비 100만 배 빠름)
기능 De novo 펩타이드 바인더 설계 → Boltz-2/AlphaFold2 3D 구조 예측 → 열역학적 결합 분석
접근성 무료 티어 제공, 학술 요금제 $299/월, GPU 병렬 처리
입력 수용체 3D 구조만 있으면 브라우저에서 바로 실행

KAIST × 데이비드 베이커 — AI 기반 특이적 결합 단백질 설계 성공

KAIST 이규리 교수와 노벨화학상 수상자 데이비드 베이커 교수가 공동으로, AI를 활용해 특정 화합물을 정밀 인식하는 단백질을 scratch에서 설계하는 데 성공했다. 결과는 Nature Communications에 게재됐다.
  • 핵심 성과 — 코르티솔을 포함한 6종 화합물 대상 결합 단백질 설계 성공, 미국 가특허 출원
  • 원리 — 단백질-리간드 상호작용을 정확히 반영하는 AI 모델 개발 → 화학적 유도 이합체(chemically induced dimer) 바이오센서 구현
  • 응용 — 질병 조기 진단, 신약 개발, 환경 오염물질 탐지 센서
  • 지원 — 과기정통부 InnoCORE 이니셔티브 (AI-CRED 혁신신약연구단)

생성형 AI 기반 펩타이드 설계 — RSC 2026 종합 리뷰

Chemical Communications(RSC, 2026)에 발표된 리뷰가 생성형 AI 기반 펩타이드 치료제 설계의 현황을 종합 정리했다.
분류주요 도구/접근법특징
구조 예측 AlphaFold, ColabFold, RoseTTAFold, ESMFold 각기 다른 펩타이드 모델링 장점 보유
결합 예측 Transformer, Graph Neural Network, 단백질 언어모델 장거리 의존성 포착, 전이학습 활용
생성 모델 RFdiffusion, DiffPepBuilder, Flow Matching, RL 백본 생성 → 고리화, 비천연 아미노산 지원
임상 번역: 2020~2024년 펩타이드 신약 승인 건수 지속 증가 (대사질환, 종양, 의료 영상). 그러나 고품질 학습 데이터 부족, 면역원성·약동학 모델링 한계, 제조 가능성 불확실성은 지속적 도전 과제.
