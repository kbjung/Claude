34가지 핵심 메시지

1엔드투엔드 전환
AI 펩타이드 설계는 개별 도구 → 엔드투엔드 파이프라인으로 진화. 단일 모델 실험이 아닌 통합 워크플로 시대.
2모델 방식 다양화
MLM, Diffusion, Flow-matching, Graph VAE 등 공존. "하나의 정답" 모델은 아직 없음 — 사용 케이스별 선택 필요.
3Physics-informed ML
물리 정보 결합이 새로운 표준. 순수 데이터 기반 → 물리 법칙을 학습에 반영하는 방향.
4Agentic AI + CoT
다음 프론티어. LLM이 전체 파이프라인을 추론하며 오케스트레이션.

4생성 모델 분류 (Generation 단계 핵심)

계열원리대표 모델장점한계
MLM
(Masked LMMasked Language Model — 입력 문장(서열)에서 일부 단어를 가리고 그 자리를 주변 맥락으로 맞히도록 학습하는 양방향 언어 모델)		 빈칸 채우기, 양방향 ★ PepMLM 빠름, 타겟 조건부타겟 단백질 서열을 입력 조건으로 주면 그 타겟에 특화된 펩타이드를 생성하도록 학습된 방식 긴 서열엔 불리
DiffusionDiffusion Model — 데이터(구조)에 노이즈를 점차 추가했다가 거꾸로 제거하는 과정을 학습. 이미지 생성의 Stable Diffusion과 같은 원리를 단백질 구조에 적용
(구조 기반)		 노이즈 제거로 구조 생성 RFdiffusion, RFpeptides 드노보 설계De novo = 라틴어 "처음부터 / 완전히 새롭게". 기존 자연 단백질·펩타이드 서열을 참고하지 않고 원하는 기능/형태만 지정하면 완전히 새로운 서열·구조를 만들어내는 설계. 자연에 없는 분자도 생성 가능 VRAM·시간 부담
Flow-matchingFlow Matching — 확산 모델의 대안. 노이즈→데이터 변환을 "단계적 경로(flow)"로 학습해 샘플링 속도를 대폭 단축한 생성 방식 연속 경로 학습 PepFlow 빠른 샘플링 학습 데이터 민감
Graph + Latent Diffusion 그래프 인코딩Graph Encoding — 원자를 노드(점), 결합을 엣지(선)로 표현해 분자를 그래프 구조로 다루는 방식. 그래프 신경망(GNN)으로 처리 + 잠재공간 확산Latent Space Diffusion — 고차원 원본 데이터가 아닌 저차원 "잠재 표현"에서 확산 과정을 수행해 속도·효율을 높임 PepGLAD 선형·고리형Cyclic Peptide — 펩타이드 양쪽 끝이 연결되어 고리를 이룬 형태. 일반 선형 펩타이드보다 체내 안정성·표적 친화도가 높아 신약 개발에서 주목받음 통합 구현 복잡
AF2 HallucinationAI가 "원하는 구조/기능을 가진 분자"가 되도록 입력 서열을 거꾸로 최적화하는 역설계 기법. AlphaFold2의 예측 신뢰도(ipTM)를 목적함수로 삼아 서열을 점진적으로 수정 AF2로 역설계 BindCraft ipTMinterface predicted TM-score — AlphaFold2/3이 예측한 두 사슬 간 계면(interface) 구조의 신뢰도 점수. 0~1 범위, 높을수록 결합 인터페이스 예측이 확실 기반 최적화 단일 지표 의존

📖 이 표에서 쓰인 핵심 용어 정리

드노보(De novo) 설계
라틴어로 "처음부터 완전히 새롭게"라는 뜻. 자연 단백질/펩타이드 서열을 참고하지 않고, 목표 기능(특정 단백질에 결합하기 등)만 조건으로 주어 AI가 완전히 새로운 분자를 생성하는 설계 방식. 자연에 존재하지 않는 분자도 만들 수 있어 신약 개발에서 강력한 도구로 떠오름.
MLM (Masked Language Model)
"빈칸 채우기" 방식으로 학습하는 AI. 문장/서열의 일부를 가리고 맥락으로 맞히게 훈련. BERT 계열이 대표적이며, 단백질 서열에 적용한 것이 ESM-2·PepMLM.
Diffusion Model
이미지에 노이즈를 점차 주입했다가 거꾸로 제거하는 과정을 학습하는 생성 모델. Stable Diffusion, DALL-E 등이 이 방식이며, 단백질 설계에는 RFdiffusion이 대표적.
고리형 펩타이드(Cyclic peptide)
펩타이드 양 끝이 서로 연결되어 고리 모양을 이룬 분자. 선형보다 체내 안정성·표적 결합력이 높아 신약 개발에서 주목.
ipTM
interface predicted TM-score. AlphaFold가 예측한 두 단백질(혹은 단백질-펩타이드) 간 결합 인터페이스의 신뢰도 점수. 0~1, 높을수록 좋음. 우리 시스템 FinalScore에 10% 반영.
Hallucination (역설계)
AI가 "이런 구조/기능이 나오는 서열을 만들어라"는 조건 아래 입력 서열을 거꾸로 최적화해가는 기법. AF2의 ipTM을 목적함수로 삼아 서열을 바꿔가며 원하는 결합을 유도.
타겟 조건부 생성(Target-conditioned generation)
표적 단백질 서열을 입력으로 주면, 그 표적에만 특화된 펩타이드를 생성하도록 학습된 방식. PepMLM의 핵심 특징.
