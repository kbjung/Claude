Anthropic, Claude Opus 4.7 출시

4월 16일 정식 공개. SWE-bench Verified 87.6%, GPQA 94.2%로 Opus 4.6 대비 12/14개 벤치마크 우위. 비전 해상도 3.3배 향상(1,568px → 2,576px). 사이버보안 자동 차단 기능과 Cyber Verification Program 신설.
  • 가격 유지: 입력 $5 / 출력 $25 (1M 토큰)
  • API, Bedrock, Vertex AI, Foundry 동시 지원
Anthropic LLM 코딩
🔮

Claude Mythos 5 — 10조 파라미터 시대 개막

사상 최초의 10T 파라미터 모델. MoE 아키텍처로 실제 활성 파라미터는 약 0.8~1.2T. 사이버보안·학술 추론·코딩 분야에서 Opus 4.6을 압도적으로 상회. 현재 제한된 얼리액세스 테스트 중이며 일반 공개 시점은 미정.
⚠️ 3월 말 Anthropic 내부 CMS 오설정으로 ~3,000개 자산이 유출되며 존재가 공개됨. 내부 코드명 "Capybara".
Anthropic 10T 파라미터 미공개

Google, Gemma 4 오픈소스 공개

4월 2일 출시. E2B(2B), E4B(4B), 26B MoE, 31B Dense 4개 변형 모델. Apache 2.0 완전 오픈 라이선스로 상업적 사용 무제한. 멀티모달(텍스트+이미지+오디오), 최대 256K 컨텍스트 윈도우, 라즈베리파이부터 고성능 GPU까지 지원.
  • 31B Dense 모델이 400B급 경쟁 모델과 동등한 벤치마크
  • HuggingFace, Ollama, Kaggle 배포
Google DeepMind 오픈소스 멀티모달
🐉

DeepSeek V4 — 1T MoE, 화웨이 칩 가동

1T 파라미터 MoE 모델(활성 ~37B). Apache 2.0 오픈 웨이트. 듀얼 RTX 4090에서 구동 가능. 4월 4일 로이터 확인: 화웨이 Ascend 950PR 칩으로 운영되는 최초의 프론티어 모델. 훈련 비용 약 $5.2M으로 극히 저렴.
  • Manifold-Constrained Hyper-Connections, Engram Memory, Sparse Attention 3대 혁신
  • SWE-bench 81%, 컨텍스트 100만 토큰
DeepSeek 화웨이 칩 오픈소스
