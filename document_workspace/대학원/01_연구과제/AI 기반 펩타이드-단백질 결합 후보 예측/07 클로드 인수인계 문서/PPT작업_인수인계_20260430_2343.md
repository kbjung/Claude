## 4. 작업 수행 방법

### 사전 준비
1. `pptx/SKILL.md` 스킬 파일 읽기
2. 사용자가 제공한 새 PPT 파일을 unpack하여 XML 구조 파악
3. 기존 2355 PPT의 레이아웃 패턴 확인 (필요 시)

### 작업 절차
1. **PPT unpack**: ZIP으로 변환 → XML 추출
2. **슬라이드 매핑**: 빈 복제 슬라이드가 어느 구성안 슬라이드에 대응하는지 확인
3. **내용 삽입**: 구성안의 영문 프롬프트/한글 내용을 참조하여 XML 텍스트 노드에 삽입
4. **repack**: XML → ZIP → .pptx로 재조립
5. **검증**: Python-pptx로 각 슬라이드 텍스트 추출하여 내용 확인

### 주의사항
- **구성안에서 "이전 버전과 동일"인 슬라이드는 건드리지 않는다**
- 슬라이드 내용은 **영어**로 작성 (구성안의 영문 프롬프트 참조)
- 한글 내용은 내용 이해용 참고자료이며, PPT에는 영어만 들어간다
- 파일명 규칙: `생체재료공학_특론_파이프라인_발표자료_YYYYMMDD_HHMM.pptx` (KST 기준)
- 새 파일로 저장 (덮어쓰기 금지)
