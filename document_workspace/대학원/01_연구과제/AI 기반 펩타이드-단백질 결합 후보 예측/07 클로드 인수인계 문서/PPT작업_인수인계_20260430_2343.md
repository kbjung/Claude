## 5. 작업 절차

1. `pptx/SKILL.md` + `editing.md` 스킬 파일 읽기
2. PPT unpack (ZIP → XML)
3. 기존 슬라이드 XML 구조 분석 (복제 원본의 레이아웃 파악)
4. 슬라이드 복제 (XML 파일 복사 + Content_Types/rels 업데이트)
5. 복제된 슬라이드 내용 비우고 구성안 영문 내용 삽입
6. 기존 슬라이드 2개 수정 (#11, #23)
7. 슬라이드 순서 조정 (presentation.xml)
8. Repack → 새 파일로 저장
9. 검증 (markitdown + 이미지 변환 후 시각 확인)

## 6. 기존 PPT 디자인 패턴

| 슬라이드 유형 | 레이아웃 특징 |
|-------------|-------------|
| 표지 | 중앙 큰 제목 + 부제 + 아이콘 |
| 목차 | 넘버링(01~07) + 파트 제목 + 한줄 부제 |
| 일반 | 좌상단 섹션 태그 + 제목 + 카드형 2~4칸 |
| STEP | INPUT→PROCESS→OUTPUT + 오른쪽 원리/특징 카드 |
| 비교 | 좌우 2단 대비 + 하단 핵심 차이 카드 |
| Appendix | "APPENDIX \| Technical Details" 태그 + 상세 카드 4칸 |
| 공통 | 슬라이드 번호 우하단, 아이콘/이미지 활용 |

## 7. 주의사항

- **구성안에서 "이전 버전과 동일"인 슬라이드는 절대 건드리지 않는다**
- PPT 내용은 **영어**로만 작성 (구성안의 한글 내용은 참고용)
- 파일명: `시스템_설명서_YYYYMMDD_HHMM.pptx` (KST 기준)
- 새 파일로 저장 (덮어쓰기 금지)
- 구성안 전문은 `01 파이프라인 설명서/시스템_설명서_구성안_20260421_1740.md` 참조
