## 1. BMPR1A 벤치마크 연구 상세 내용

**논문**: *AI-guided design of candidate BMPR1A-binding peptides for cartilage regeneration: a multi-tool computational benchmarking study* (bioRxiv, 2026.03.25)

### 연구 설계
- BMPR1A(연골 재생 관련 수용체)의 BMP-2 결합 부위를 표적으로 하는 펩타이드를 AI로 설계
- 4개 도구 × 48개 = **총 192개 후보 펩타이드** 생성
- **완전히 in silico 평가** — 실험적 검증 없음

### 사용된 4개 AI 도구

| 도구 | 방법론 |
|------|--------|
| RFdiffusion | 확산 모델, 구조 기반 de novo 설계 |
| RFpeptides | 확산 모델(RFdiffusion 파생), 고리형 펩타이드 전용 |
| PepMLM | 마스킹 언어 모델(ESM-2 기반), 서열 기반 생성 |
| BindCraft | 구조 기반 설계, AlphaFold2 활용 |

### 4가지 평가 지표 (교차 검증)

| 지표 | 도구 | 의미 |
|------|------|------|
| ipTM | ColabFold | 복합체 구조 예측 신뢰도 |
| dGseparated | PyRosetta | 물리 기반 결합 에너지 (REU) |
| ΔG | FoldX | 경험적 결합 자유에너지 (kcal/mol) |
| Contact Recapitulation | 자체 분석 | BMP-2 실제 결합 잔기 30개 중 재현율 → **결합 부위 특이성** |

### 주요 결과
- 최우수 후보: PepMLM 설계 15-mer `pepmlm_L15_0026`
  - dGseparated = −45.9 REU, ΔG = −19.4 kcal/mol, Contact recapitulation = 11/30
- BindCraft: ipTM 최고(0.81)지만 Contact Recapitulation 평균 0.224로 낮음 → 엉뚱한 부위 결합 가능성
- 최종 54개 후보로 압축

### 핵심 교훈
단일 지표(예: ipTM) 만으로는 부족 — 결합 에너지가 좋아도 **원하는 부위에 결합하는지** 별도 검증 필요

## 2. pepbind06 파이프라인과의 비교

| 항목 | pepbind06 | BMPR1A 연구 |
|------|-----------|-------------|
| 펩타이드 생성 | PepMLM 1종 | 4종 병렬 |
| 구조 예측 | ColabFold (AF-Multimer) | ColabFold (동일) |
| 구조 정제 | OpenMM (에너지 최소화 + MD 100ps) | **없음** → 우리가 앞섬 |
| 결합 에너지 | Vina score_only + PRODIGY | PyRosetta + FoldX 2종 교차 |
| 결합 부위 검증 | **없음** | Contact Recapitulation |
| 자동 재시도 | GBSA>100 시 최대 3회 | **없음** → 우리가 앞섬 |
| FinalScore | PRODIGY 60% + Vina 30% + ipTM 10% | 4지표 복합 순위 |

### 도출된 개선 아이디어
- **Contact Recapitulation 도입 검토**: 구현 비용 낮고, 결합 부위 특이성 검증 효과 직접적
  - 현재 진행 중인 ADCP 검토와 병행 가능
  - FinalScore 반영 또는 필터 조건으로 활용 가능
- PyRosetta dGseparated 추가 검토 (Vina 보완 목적)

## 3. PepMLM 논문 실험적 검증 확인

**논문**: *Target sequence-conditioned design of peptide binders using masked language modeling*, Nature Biotechnology (2025년 8월 정식 게재)  
> ⚠️ arXiv 프리프린트(2023)에는 없었고, 저널 게재 시 wet lab 실험 추가됨

### 실험 1 — ELISA (체외 결합 확인)
- 표적: NCAM1(AML 마커), AMHR2(PCOS 관련 수용체)
- PepMLM vs RFdiffusion 각 4개씩 비교 스크리닝

| 표적 | PepMLM | RFdiffusion |
|------|--------|-------------|
| NCAM1 | 4개 모두 결합 (~30 nM, p=0.0051) | 중간 수준 결합 |
| AMHR2 | 4개 모두 결합 (~60 nM) | **거의 결합 없음** |

→ 구조 정보 없는 표적에서 PepMLM의 강점이 실험으로 증명됨

### 실험 2 — 세포 내 단백질 분해 실험
- 방법: PepMLM 펩타이드를 E3 유비퀴틴 리가제(CHIP△TPR)에 융합 → 세포 내 표적 단백질 분해
- 표적: MSH3·mHTT(헌팅턴병), Nipah/Hendra/HMPV 바이러스 인단백질
- 결과: **40% 히트율**, 표적 단백질 25~100% 감소

### 파이프라인 관련 의미
- PepMLM이 in silico → 실험까지 결합을 증명 → 우리 파이프라인 고점수 후보의 실험 유효성에 대한 문헌적 근거 확보
- PepMLM은 3D 구조 없이 서열만으로 설계 가능 → 구조 불분명한 표적(intrinsically disordered protein 등)에도 적용 가능
