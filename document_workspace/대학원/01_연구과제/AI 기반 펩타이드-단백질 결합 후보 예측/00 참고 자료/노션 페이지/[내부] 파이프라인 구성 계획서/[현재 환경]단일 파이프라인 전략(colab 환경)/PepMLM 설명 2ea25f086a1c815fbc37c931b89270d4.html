PepMLM 설명

  • 출처 : https://www.nature.com/articles/s41587-025-02761-2

내용 정리(범)

  • 실제 단백질-펩타이드 데이터를 훈련 시킨 모델.
  • 단백질의 구조나 모양같은 물리적 구조를 실제 단백질 서열 데이터를 학습하여 간접적으로 유추하는 방식
    • 단백질들의 특정 서열들이 주로 안쪽에 위치하거나 굽어서 접근이 어려운 위치에 존재한다면, 펩타이드와 결합하지 못하는 서열로 학습이되어, 결합 펩타이드와 발견될 확률이 매우 낮거나 거의 없게됨
    • 물리적인 구조적 제약을 학습 데이터 내의 통계적 패턴으로 간접적으로 해석하고 적용하는 방식
  • 단백질의 특정 타겟 지정해서 진행하는 방식은 아님
  • 학습 데이터는 PepNN 훈련 세트와 Propedia 데이터세트라는 기존의 펩타이드-단백질 결합 데이터
    • 비슷한 불필요한 데이터 제거, 유사성이 높은 단백질(80% 임계값) 수동 제거
    • 최종 10,203개. 10,000개 모델 훈련, 203개 모델 테스트.
    • 최대 길이 제한 : 펩타이드 서열 50, 단백질 서열 500

PepMLM: 새로운 펩타이드 결합체 설계 기술

PepMLM은 마스크드 언어 모델링(masked language modeling)을 활용하여 ESM-2 단백질 언어 모델을 미세 조정함으로써, 새로운 선형 펩타이드 결합체를 설계하는 기술입니다. 기존의 단백질 결합체 설계는 안정적인 3차원 구조에 의존했지만, PepMLM은 구조적 정보 없이

표적 단백질 서열만으로 결합체를 설계할 수 있다는 장점이 있습니다.

주요 작동 원리와 특징

  • 마스킹 전략: PepMLM은 표적 단백질 서열의 C-말단에 결합 펩타이드 서열을 위치시키고, 이 결합 부위를 마스킹하는 전략을 사용합니다. 이를 통해 ESM-2 모델은 마스킹된 펩타이드 영역 전체를 재구성하도록 훈련됩니다.
  • 낮은 퍼플렉시티(Perplexity): PepMLM으로 설계된 펩타이드는 실제 검증된 펩타이드-단백질 서열 쌍과 유사하거나 더 낮은 퍼플렉시티 값을 보이며, 이는 모델이 펩타이드 서열을 정확하게 예측하고 있음을 의미합니다. 퍼플렉시티 값은 결합 상호작용을 반영하며, 낮을수록 안정적인 결합 가능성이 높음을 시사합니다.
  • 우수한 성능: 구조 기반 설계 모델인 RFdiffusion과 비교했을 때, PepMLM은 in silico(컴퓨터 모의실험)에서 더 높은 적중률을 보였습니다. 특히, PepMLM은 훈련 데이터셋과 서열 유사성이 낮은 새로운 표적에 대해서도 일관되게 낮은 퍼플렉시티와 높은 결합 예측 능력을 보여 일반화 성능이 뛰어납니다.

실험적 검증 및 응용 분야

  • 특이적 결합 및 단백질 분해: PepMLM으로 설계된 펩타이드는 실험적으로 암 및 생식 관련 표적(NCAM1, AMHR2 등)에 대한 서열 특이적 결합 능력을 보였습니다. 또한, 헌팅턴병(Huntington's disease)과 같은 다양한 질병 관련 단백질을 **표적 단백질 분해(TPD)**하는 데 효과적임이 입증되었습니다.
  • 치료제 개발 촉진: PepMLM은 구조 정보가 없는 '약물 표적화가 어려운(undruggable)' 단백질에 대한 치료용 결합체 후보 물질을 설계할 수 있게 함으로써, 신약 개발 분야에 폭넓게 활용될 수 있습니다.

마스크드 언어 모델의 장점

1. 양방향 문맥 이해

MLM은 문장의 중간에 있는 단어를 예측하기 위해 단어의 앞뒤 모든 문맥을 동시에 고려합니다. 기존의 단방향 언어 모델(예: GPT)이 '다음 단어'만 예측하는 것과 달리, MLM은 문장의 전체적인 의미를 파악하는 데 훨씬 유리합니다. 이를 통해 단백질 서열처럼 앞뒤 서열 정보가 모두 중요한 경우에 더욱 효과적으로 단백질의 기능적, 구조적 특성을 학습할 수 있습니다.

2. 효율적인 사전 학습

MLM은 대량의 레이블 없는(unlabeled) 데이터로 사전 학습을 진행할 수 있습니다. 단백질 언어 모델의 경우, 수십억 개의 단백질 서열 데이터에서 임의의 부분을 가리고 예측하는 단순한 작업만으로도 모델을 강력하게 만들 수 있습니다. 이는 복잡한 구조적 정보나 기능적 주석 데이터가 부족한 상황에서도 모델을 훈련시킬 수 있게 해줍니다.

3. 세부 작업에 대한 미세 조정(Fine-tuning) 용이성

MLM을 통해 사전 학습된 모델은 특정 작업(예: 펩타이드 결합체 설계)에 맞게 미세 조정하기 용이합니다. 이미 방대한 단백질 서열의 문맥적 관계를 학습했기 때문에, 적은 양의 레이블링된 데이터만으로도 새로운 작업에 빠르게 적응할 수 있습니다. PepMLM이 ESM-2(MLM 기반 모델)를 미세 조정하여 펩타이드 결합체 설계에 성공적으로 적용된 것이 대표적인 예입니다.

4. 생성 능력

언뜻 보면 MLM은 '빈칸 채우기' 모델처럼 보이지만, 이를 반복적으로 적용하면 새로운 서열을 효과적으로 생성할 수 있습니다. 예를 들어, PepMLM 모델은 표적 단백질 서열의 끝에 펩타이드 결합 부위 전체를 마스킹하고, 이 빈 영역을 예측함으로써 완전히 새로운 펩타이드 서열을 생성합니다. 이는 단순히 기존 서열을 완성하는 것을 넘어, 문맥에 맞는 새로운 서열을 창의적으로 만들어내는 능력으로 이어집니다.

인코더 전용 모델 설명

인코더 전용 모델(Encoder-only model)은 입력 데이터를 이해하고 의미 있는 표현으로 변환하는 데 특화된 모델입니다.

간단히 말해, 인코더는 입력된 문장이나 서열의 '맥락(context)'을 파악하는 역할을 합니다. 마치 누군가에게 복잡한 내용을 전달받고 그 핵심 내용을 요약해서 머릿속에 저장하는 것과 유사합니다.

주요 특징과 작동 방식

  • 양방향 문맥 분석: 인코더 전용 모델은 입력 데이터의 전체를 한꺼번에 보고 각 요소(단어, 아미노산 등)가 다른 요소와 어떻게 연관되는지 파악합니다. 즉, 문장의 앞과 뒤 모든 정보를 동시에 고려하여 문맥적 의미를 파악합니다.
  • 대표적인 모델: BERT, ESM-2 등이 인코더 전용 모델에 속합니다. 이들은 주로 마스크드 언어 모델링(MLM) 방식을 통해 훈련됩니다. 입력 데이터의 일부를 가린 뒤, 주변의 문맥을 바탕으로 가려진 부분을 예측하는 훈련을 반복합니다.
  • 활용: 이러한 모델은 주로 분류, 요약, 문맥 이해, 검색 등 입력 데이터를 분석하고 이해하는 작업에 강점을 보입니다.

인코더-디코더 모델과의 차이

  • 인코더-디코더 모델: 이 모델은 인코더와 디코더가 결합된 형태입니다. 인코더는 입력 문장을 이해하고, 디코더는 인코더가 파악한 내용을 바탕으로 새로운 문장을 생성합니다. 번역(예: 영어 -> 한국어)이나 요약과 같은 작업에 주로 사용됩니다.
  • 인코더 전용 모델: 이 모델은 디코더 없이 인코더만으로 작동합니다. 따라서 새로운 서열을 '생성'하는 것보다는, 주어진 서열의 '특성'이나 '의미'를 파악하는 데 더 최적화되어 있습니다. 다만, PepMLM의 사례처럼 인코더 전용 모델도 특정 목적에 맞게 훈련하면 생성 작업에 활용될 수 있습니다. 이 경우, 모델은 특정 조건(표적 서열)에 맞춰 빈칸(마스킹된 펩타이드 서열)을 채우는 방식으로 새로운 서열을 생성하게 됩니다.

ESM-2 모델의 특징

ESM-2(Evolutionary Scale Modeling-2)는 단백질 언어를 이해하도록 훈련된 언어 모델입니다. 자연어 처리(NLP) 분야에서 발전한 어텐션(attention) 메커니즘을 기반으로 구축되었으며, 수십억 개의 아미노산 서열로 구성된 대규모 단백질 데이터셋을 학습했습니다.

주요 특징은 다음과 같습니다.

  • 인코더 전용(Encoder-only) 모델: ESM-2는 단백질 서열 내의 공진화(co-evolutionary) 패턴을 식별하는 인코더 전용 모델로 설계되었습니다.
  • 마스크드 언어 모델링(MLM): 훈련 시, 단백질 서열의 일부를 가린 뒤(마스킹), 주변 서열의 문맥을 바탕으로 가려진 부분을 예측하는 마스크드 언어 모델링 방식을 사용합니다.
  • 단백질 서열 표현: ESM-2는 단백질 서열의 핵심적인 물리화학적 특성을 정확하게 포착하며, 단백질 정보를 임베딩(embedding)하는 데 효과적입니다.

활용 분야

ESM-2는 단백질 서열 정보를 깊이 있게 이해하므로 다양한 분야에 활용됩니다. 항체 설계, 새로운 단백질 생성, 단백질 구조 예측 등 여러 애플리케이션의 기반 모델로 사용됩니다. 특히 PepMLM 모델은 ESM-2를 미세 조정하여 단백질 결합체 설계에 특화시킨 사례라고 할 수 있습니다.
