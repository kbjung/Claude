## 2. IL-13Rα2란?

| 항목 | 내용 |
|------|------|
| 정식 명칭 | Interleukin-13 Receptor Subunit Alpha-2 |
| 위치 | 암세포 표면 수용체 (세포막 단백질) |
| 특징 | 정상 세포엔 거의 없고, **암세포에만 과발현** |
| 신호전달 | Jak/STAT 신호 없음 → decoy receptor로 기능 |
| UniProt | Q14627 |

### 쉬운 비유
> 암세포 표면에 꽂혀있는 **특수 자물쇠**
> → 이 자물쇠에 맞는 펩타이드(열쇠)를 만들어 암세포만 선택적으로 공격

### 주요 적응증
- 교모세포종(뇌암) — 연구 가장 활발
- 췌장암, 난소암, 중피종 등 고형암

## 3. 핵심 PDB 구조

| PDB ID | 내용 | 용도 |
|--------|------|------|
| **3LB6** ⭐ | IL-13 / IL-13Rα2 복합체 결정구조 | **1차 타겟** — 결합 인터페이스 분석 |
| **3BPO** | IL-13 / IL-4Rα / IL-13Rα1 삼중 복합체 | 비교 참고 (Rα1 vs Rα2 차이 확인) |

### 3LB6 활용 포인트
- IL-13Rα2가 IL-13Rα1 대비 결합 친화도 **10,000배 높음** (femtomolar 수준)
- 결합 인터페이스가 크고 명확 → 펩타이드 docking 타겟으로 최적
- **apo 구조(수용체 단독) 없음** → 3LB6에서 수용체 chain만 추출해 사용

### 실무 워크플로우
```
3LB6 다운로드
  → IL-13Rα2 chain 단독 추출
  → 결합 인터페이스 잔기 확인 (hotspot residues)
  → 펩타이드 docking 타겟 포켓으로 활용
```

**RCSB 링크:** https://www.rcsb.org/structure/3LB6
