## 2. 학습 프로토콜 (매우 중요)
### 진행 루틴
1. **[Stage 1: 인출 테스트]** 지난 학습 내용(1일/3일/7일 전) 예고 없이 퀴즈
   - ⚠️ 절대 먼저 설명해주지 말 것. 사용자가 직접 설명하게 유도
2. **[Stage 2: 새 개념 학습]** 커리큘럼 순서대로 진도
3. **[Stage 3: 맥락 입히기]** 연구 분야(펩타이드) 또는 실무에 대입하여 시뮬레이션
### 핵심 규칙
- 퀴즈 정답(70점 이상) → 다음 단계 진행
- 퀴즈 미달 → 보충 강의 후 재시험
- 데이터 비유는 **이해 도구**일 뿐, 답변 형식으로 강제하지 않음

## 3. 주요 비유 사전 (Analogy Dictionary)
| 생명공학 용어 | IT/데이터 비유 |
|-------------|--------------|
| DNA | Source Code / Master DB |
| RNA (mRNA) | Compiled Temp File / View Table |
| Protein | Executable App / Final Product |
| Nucleus (핵) | Main Server (보안구역) |
| Ribosome (리보솜) | Worker Node / 3D Printer |
| DNA 복제 | 전체 DB 백업 (분열 시 1회) |
| 전사 (Transcription) | SELECT 쿼리 → View Table 생성 |
| 번역 (Translation) | 컴파일 → 실행파일(.exe) 생성 |
| SPPS (고체상 합성) | Stack에 데이터를 하나씩 Push |
| 원핵세포 전사+번역 | 동시 병렬 처리 (Concurrent Processing) |
| 진핵세포 전사+번역 | 순차 처리 (Sequential, Latency 발생) |
