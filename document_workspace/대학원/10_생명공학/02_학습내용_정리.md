## Level 1. 세포 생물학 기초
### 원핵세포 vs 진핵세포
| 구분 | 원핵세포 | 진핵세포 |
|------|---------|---------|
| 비유 | 1인 스타트업 | 대기업 |
| 핵막 | 없음 | 있음 |
| 전사+번역 | 동시 진행 (병렬) | 순차 진행 (핵 → 세포질) |
| 분열 속도 | 빠름 (대장균: 20분) | 느림 (보통 24시간) |
| 대량생산 유리 여부 | ✅ 유리 | △ 상대적으로 불리 |

**대량생산 유리한 이유:**
핵막이 없어 전사와 번역이 동시에 진행됨 → Latency 없음 → 생산 속도 빠름

## Level 2. 분자생물학 기초
### 중심 원리 (Central Dogma)
DNA → (전사) → mRNA → (번역) → Protein

| 과정 | Input | Output | 빈도 | 비유 |
|------|-------|--------|------|------|
| DNA 복제 | DNA | DNA | 세포분열 시 1회 | 전체 DB 백업 |
| 전사 | DNA (일부) | mRNA | 수시로 반복 | SELECT 쿼리 실행 |
| 번역 | mRNA | Protein | 수시로 반복 | 컴파일 → .exe 생성 |

⚠️ 핵심 오개념 주의: 전사 ≠ DNA 복제
- DNA 복제: 게놈 전체 복사 (분열 준비)
- 전사: 필요한 일부 유전자만 mRNA로 복사

## Level 3. 펩타이드 공학
### 3-1. 펩타이드 기초
- 아미노산 2~50개 → 펩타이드
- 아미노산 50개 이상 → 단백질
- 펩타이드 결합: 아미노기(-NH₂) + 카르복실기(-COOH) → 탈수 축합

### 3-2. 펩타이드 합성 (SPPS)
- 고체 수지(Resin)에 C말단 → N말단 방향으로 아미노산을 하나씩 연결
- 비유: Stack 자료구조에 데이터를 하나씩 Push

### 3-4. 펩타이드 반감기 연장 전략
**왜 짧은가?**
체내 프로테아제가 L-아미노산을 표준 타겟으로 인식 → 빠르게 분해

**전략 1: 비천연 아미노산 도입**

| 방법 | 원리 | 효과 |
|------|------|------|
| D-아미노산 사용 | 자연계는 L형만 존재, D형은 거울상 | 프로테아제 인식 불가 → 분해 저항 |
| N-메틸화 | 펩타이드 결합부에 메틸기 부착 | 결합부 보호 |

**전략 2: 환상화 (Cyclization)**
- N말단과 C말단을 연결 → 고리(Cyclic) 구조 형성
- 효과: 분해효소 접근 차단 + 구조 안정화
- 실제 사례: 옥시토신, 사이클로스포린

### 2-5. 아미노산 20종 (Level 3 진입 전 선행 학습)

**핵심 구조**: 공통 골격(H₂N-C-COOH) + 고유 R기(곁사슬) → R기가 모든 특성 결정

**4가지 분류**

| 분류 | 특성 | 아미노산 (3-letter / 1-letter) |
|------|------|------|
| 비극성 (소수성) | R기 탄화수소계, 물 싫어함 | Gly(G), Ala(A), Val(V), Leu(L), Ile(I), Pro(P), Phe(F), Trp(W), Met(M) |
| 극성 (친수성, 중성) | R기에 -OH/-SH 등, 수소결합 가능 | Ser(S), Thr(T), Cys(C), Tyr(Y), Asn(N), Gln(Q) |
| 양전하 (+) | 생리적 pH에서 R기 양성 | Lys(K), Arg(R), His(H) |
| 음전하 (−) | 생리적 pH에서 R기 음성 | Asp(D), Glu(E) |

**20종 전체 정리**

| 풀네임 | 약자 | 분류 | 핵심 역할 |
|--------|------|------|----------|
| Glycine | Gly / G | 비극성 | R기=H, 가장 작음, 구조 유연성 최대 |
| Alanine | Ala / A | 비극성 | 가장 단순한 소수성 AA, 작고 안정적 |
| Valine | Val / V | 비극성 | 분지형 소수성, β-sheet 선호 |
| Leucine | Leu / L | 비극성 | 강한 소수성, α-helix 선호, HPLC retention time 증가 |
| Isoleucine | Ile / I | 비극성 | 분지형 소수성, Val과 유사 |
| Proline | Pro / P | 비극성 | 백본과 고리 형성 → 구조 꺾임, 2차구조 파괴자 |
| Phenylalanine | Phe / F | 비극성 | 방향족 고리, 강한 소수성, HPLC 오른쪽 |
| Tryptophan | Trp / W | 비극성 | 가장 큰 AA, 강한 소수성, UV 280nm 흡수 |
| Methionine | Met / M | 비극성 | 황(-S-) 함유, 단백질 번역 시작 AA (AUG) |
| Serine | Ser / S | 극성 (중성) | -OH기, 인산화 타겟, SPPS 보호기 필요 |
| Threonine | Thr / T | 극성 (중성) | -OH기, 인산화 타겟 |
| Cysteine | Cys / C | 극성 (중성) | -SH기, S-S 이황화결합으로 구조 고정 |
| Tyrosine | Tyr / Y | 극성 (중성) | 방향족 + -OH기, 인산화 타겟, UV 280nm 흡수 |
| Asparagine | Asn / N | 극성 (중성) | 아미드기, N-글리코실화 타겟 |
| Glutamine | Gln / Q | 극성 (중성) | 아미드기, 세포 배양 주요 에너지원 |
| Lysine | Lys / K | 양전하 (+) | 가장 흔한 양전하 AA, 약물·PEG 접합 포인트, SPPS 보호기 필요 |
| Arginine | Arg / R | 양전하 (+) | 강한 양전하, 세포막 투과 펩타이드 핵심 |
| Histidine | His / H | 양전하 (+) | pH에 따라 전하 변화(pKa≈6), 효소 활성부위 자주 등장 |
| Aspartate | Asp / D | 음전하 (−) | 음전하, 효소 활성부위 자주 등장 |
| Glutamate | Glu / E | 음전하 (−) | 음전하, 신경전달물질 |

---

**펩타이드 연구 필수 7종**

| AA | 약자 | 핵심 역할 |
|----|------|----------|
| Glycine | Gly/G | R기=H, 구조 유연성 최대 |
| Proline | Pro/P | 백본과 고리 형성 → 구조 꺾임, 2차구조 파괴자 (중간에 있을 때 영향 큼) |
| Cysteine | Cys/C | S-S 이황화결합으로 구조 고정 |
| Lysine | Lys/K | 양전하, 약물·PEG 접합 포인트. SPPS 시 R기 별도 보호기 필요 |
| Arginine | Arg/R | 강한 양전하, 세포막 투과 펩타이드 핵심 |
| Aspartate | Asp/D | 음전하, 활성부위 자주 등장 |
| Phenylalanine | Phe/F | 강한 소수성, HPLC retention time 증가 |

**SPPS × 아미노산 연결**
- R기에 반응성 작용기(-NH₂, -SH, -OH) 있는 AA(Lys, Cys, Ser 등)는 SPPS 중 부반응 위험 → 별도 보호기 필요
- Gly처럼 R기=H인 경우 보호기 불필요

**HPLC × 아미노산 연결**
- Phe, Trp, Leu 많을수록 → 소수성 ↑ → 크로마토그램 오른쪽
- Arg, Lys 많을수록 → 친수성 ↑ → 크로마토그램 왼쪽
- Arg-Lys + Leu-Phe 공존 시 → 양친매성(Amphipathic) → 항균 펩타이드 전형적 구조

### 3-2 심화. SPPS 방향성 및 서열 표기
**SPPS 풀네임**: Solid Phase Peptide Synthesis (고체상 펩타이드 합성)

**합성 방향: C → N**
- Resin에 첫 번째 AA의 C말단이 고정됨
- 투입되는 AA는 항상 N말단이 Fmoc로 보호된 채 투입
- 반응 가능한 부위: 체인의 N말단(노출) ↔ 투입 AA의 C말단
- 결합 후 Fmoc 제거 → 다음 AA 투입 반복

**한 사이클 요약**
① Fmoc 제거 (N말단 노출) → ② 새 AA 투입 (N말단 Fmoc 보호 상태) → ③ C말단 결합 → 반복

**서열 표기 방향: N → C (국제 표준)**
- 리보솜 번역 방향과 일치
- 예: Ala-Gly-Phe-Lys = N말단(Ala) → C말단(Lys)
- SPPS 합성 시에는 역순 (Lys부터 Resin 고정)으로 진행

### 3-3. HPLC (High Performance Liquid Chromatography)
**장비 구성 순서**
시료 주입기 → 고압 펌프 → 컬럼 → UV 검출기 → 크로마토그램 출력

**역상 HPLC (Reverse Phase HPLC)**
- 고정상: 소수성 (기름 친화적)
- 이동상: 물 + 아세토니트릴 (수성 용매)
- 분리 기준: 펩타이드의 소수성 차이

| 소수성 강도 | 컬럼 체류 시간 | 크로마토그램 위치 |
|------------|--------------|----------------|
| 약함 | 짧음 (빨리 나옴) | 왼쪽 피크 |
| 중간 | 중간 | 가운데 피크 |
| 강함 | 길음 (늦게 나옴) | 오른쪽 피크 |

**크로마토그램 해석**
- 피크 위치 = 물질의 소수성
- 피크 면적 = 물질의 함량(순도)
- 목표 펩타이드 피크 구간만 수집 → 정제 완료

⚠️ SPPS 합성 후 HPLC 정제가 바로 다음 단계

### 3-5. 분취 vs 분석 HPLC

| 구분 | 분석 HPLC (Analytical) | 분취 HPLC (Preparative) |
|------|----------------------|------------------------|
| **목적** | 순도·성분 확인 | 목표 물질 대량 수집 |
| **시료량** | 수 μg ~ mg | 수십 mg ~ g |
| **컬럼 직경** | 좁음 (4.6mm) | 넓음 (10~50mm+) |
| **유속** | 느림 (1 mL/min) | 빠름 (10~100 mL/min) |
| **결과물** | 크로마토그램 (데이터) | 분리된 순수 펩타이드 (물질) |

**실험 순서**: 분석 HPLC 먼저 → 피크 위치 확인 → 분취 HPLC로 해당 구간 수집

**분획 수집 (Fraction Collection)**: 컬럼 출구 용출액을 시간대별로 튜브에 나눠 수집 → 목표 피크 구간 분획만 합쳐 농축

**실제 순도 기준**: SPPS 직후 60% 순도는 흔한 수치. 분취 HPLC 정제 후 목표 >95%
