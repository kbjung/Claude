# Layer 2 — 전략 프리셋: 플랫폼 투트랙 (피부암 + 탈모 치료제)

> **적용 시점**: 경피 전달 펩타이드 플랫폼의 다적응증 확장성을 강조하는 전략 사용 시 (창업선도대학, 예비창업패키지 등 대학 기반 창업 지원사업)
> **사용법**: Layer 1(코어 팩트시트) + 본 파일 + Layer 3(스타일 가이드) 조합으로 문서 작성
> **기반 문서**: 창업계획 요약표 (2025.11 기제출 버전)

---

## 1. 전략 방향 요약

| 항목 | 내용 |
|------|------|
| 전략명 | 경피 전달 펩타이드 플랫폼 기반 투트랙 (피부암 + 탈모 치료제) |
| Track A | 피부암(흑색종) 국소 치료제 → 동물 모델에서 치료 효능 확인, 논문 게재 완료 |
| Track B | 탈모 치료제 (국소 Finasteride 복합제) → 동물 모델에서 효능 확인(정상 생쥐), 제형 최적화 예정 |
| 핵심 논리 | 동일 경피 전달 펩타이드 플랫폼으로 2개 적응증(피부암·탈모)에 동시 접근, 플랫폼 범용성 입증 + 적응증별 사업화 경로 확보 |
| 화장품 위치 | 독립 트랙이 아닌 **사업화 초기 매출 수단** (펩타이드 단독 두피케어 제품) |

---

## 2. 제품 포지셔닝

### Track A: 피부암(흑색종) 국소 치료제

| 항목 | 내용 |
|------|------|
| 제품 유형 | 경피 전달 펩타이드 기반 항암 단백질 국소 치료제 |
| 핵심 성분 | 펩타이드 운반체 + 항암 단백질 |
| 적응증 | 피부암(흑색종) |
| 개발 단계 | 동물 모델에서 치료 효능 확인 (Imiquimod 대비 우수한 치료능, 건선 부작용 미발견) |
| 논문 근거 | Adv. Funct. Mater. 2025, 35(2), 2412160 (IF 19.0) |
| 규제 경로 | 전문의약품 (비임상 → 임상) |
| 핵심 차별성 | 비침습적 국소 투여로 기존 전신 항암치료 대비 부작용 최소화 가능성 |

### Track B: 탈모 치료제 (국소 Finasteride 복합제)

| 항목 | 내용 |
|------|------|
| 제품 유형 | 처방 기반 국소 Finasteride 복합제 |
| 핵심 성분 | 이중기능 펩타이드 운반체 + Finasteride |
| 적응증 | 안드로겐성 탈모(AGA) |
| 개발 단계 | 동물 모델에서 효능 확인(정상 생쥐), 제형 최적화 예정 |
| 핵심 수치 | 경구 대비 40배↓ 투여량, 경쟁 국소 대비 4.6배↓, Minoxidil 동등 이상 효능 |
| 규제 경로 | 전문의약품 (비임상 → 임상 → 허가) |
| 핵심 차별성 | 전달 + 항염 이중기능(carrier-free), 저용량으로 전신 노출 최소화 |

### 화장품 (사업화 초기 매출 수단)

| 항목 | 내용 |
|------|------|
| 제품 유형 | 펩타이드 단독 기능성 두피케어 제품 (leave-on 앰플/토닉) |
| 핵심 성분 | 펩타이드 단독 (약물 미포함) |
| 포지셔닝 | 의약품 개발 장기 주기 동안 초기 매출 확보 수단 |
| 규제 경로 | 기능성 화장품 또는 일반 화장품 |
| 근거 | 펩타이드 단독으로 유의미한 모발 성장 촉진 효능 확인 (전임상 데이터) |
| 제조 전략 | OEM/ODM (Bio FD&C 등) |

> ⚠️ 화장품은 독립 트랙으로 강조하지 않고, "사업화 초기 매출 확보" 또는 "플랫폼 시장 검증 수단" 수준으로 서술

## 3. 플랫폼 투트랙 시너지 구조

```
          경피 전달 펩타이드 플랫폼 (공통 핵심기술)
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
   Track A: 피부암          Track B: 탈모
   (항암 단백질 전달)       (Finasteride 전달)
   - 동물 모델 효능 확인    - 전임상 효능 확보
   - 논문 검증 완료         - 제형 최적화 예정
        │                       │
        └───────────┬───────────┘
                    ▼
           플랫폼 범용성 입증
           (적응증 확장 가능성)
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
   화장품 (초기 매출)     추가 적응증 확장
   - 펩타이드 단독        - 아토피/염증성 피부질환
   - OEM/ODM              - DKK1 억제 펩타이드
```

**플랫폼 논리:**
- 동일 펩타이드 운반체가 항암 단백질(피부암)과 Finasteride(탈모)를 각각 경피 전달 → 플랫폼 범용성 입증
- 두 적응증에서의 검증 데이터가 추가 적응증 확장의 기술적 근거로 작용
- 화장품은 의약품 개발 장기 주기를 보완하는 초기 사업화 수단

## 4. 사업모델 구조

### 4-1. 수익모델

| 수익원 | 관련 트랙 | 시기 | 설명 |
|--------|----------|------|------|
| 기능성 화장품 매출 | 화장품 | 초기 | 펩타이드 단독 두피케어 제품 OEM/ODM 생산·판매 |
| 기술이전/라이선스 | Track A·B | 중기 | 적응증별 기술 라이선스아웃, 마일스톤+로열티 |
| 공동개발 | Track B | 중기 | 대형 제약사 공동개발 (대웅제약 등), 개발비 분담 + 공동 IP |
| 플랫폼 라이선스 | 양 트랙 | 중장기 | 경피 전달 플랫폼 자체의 기술 라이선스 (다적응증) |
| 원료 공급 (B2B) | 양 트랙 | 중기~ | 펩타이드 원료 B2B 공급 |

### 4-2. 자금 조달 계획

| 단계 | 시기 | 내용 |
|------|------|------|
| 정부 지원사업 | 초기 | 예비창업패키지 → 초기창업패키지 → 도약패키지 단계별 연계 |
| 화장품 매출 | 초기~중기 | 의약품 개발 기간 동안 자체 매출 확보 |
| 기술이전/공동개발 | 중기 | 전임상 PoC 확보 후 제약사 파트너십 |
| 단계별 자금 조달 | 중장기 | 임상 진입 전후 자금 확보 |

> ⚠️ "외부 투자 유치" 대신 "단계별 자금 조달" 표현 사용

## 5. 사업화 로드맵

### 5-1. 기본 로드맵 (사업 기간 기준)

| 단계 | 기간 | Track A (피부암) | Track B (탈모) | 화장품 |
|------|------|----------------|---------------|--------|
| 1단계 | 0~12개월 | 추가 전임상 설계 | 제형 최적화, 시제품 개발 | 제형 개발, 인체적용시험 |
| 2단계 | 12~24개월 | 전임상 효능 보완 | 전임상 효능·안전성 고도화 | 제품 출시, D2C 판매 |
| 3단계 | 24~36개월 | 비임상 독성시험 설계 | GLP 독성시험, IND 준비 | 전문 채널 확산 |

### 5-2. 후속 R&BD 계획 (요약표 기준)

| 항목 | 내용 |
|------|------|
| 1차 목표 (2년 내) | 파일럿 제형 확보 + PoC 데이터 보강 |
| 2차 목표 | 비임상 독성시험 → IND 준비 |
| 연계 사업 | 예비창업패키지 → 초기창업패키지 → 도약패키지 |

## 6. 시장 진입 전략

### 6-1. 내수 시장

| 채널 | 전략 |
|------|------|
| 화장품 (초기) | OEM/ODM 생산 → 온라인 D2C + 두피/피부과 채널 |
| 치료제 (중기) | 처방 기반 (피부과/비뇨기과/탈모전문클리닉) |

### 6-2. 해외 시장

| 단계 | 전략 |
|------|------|
| 1단계 | 화장품 D2C (미국/북미 우선), 국내 데이터 기반. FDA 승인 국소 DHT 억제제 부재한 공백 시장 |
| 2단계 | 치료제 FDA IND 경로, PCT/미국특허 기반 |
| 3단계 | 글로벌 제약사 라이선스아웃, 유럽 임상 (OPIS 협력) |

## 7. 협력 네트워크

| 구분 | 파트너 | 역할 | 관련 트랙 |
|------|--------|------|----------|
| 원료/제조 | Bio FD&C | OEM/ODM, INCI 등록 | 화장품 + Track B |
| CRO/RA (국내) | Dt&CRO | 트랙별 비임상/CMC/임상 요구사항 | Track A·B |
| 글로벌 CRO | OPIS (이탈리아) | 유럽 임상, 화상 미팅 완료(2026.02) | Track A·B |
| 인체적용/임상 | P&K 피부임상연구센터 | 화장품 임상시험 | 화장품 |
| 전임상 | 에피바이오텍 | 세포/동물 모델 효능평가 | Track A·B |
| 임상 자문 | 인하대병원 피부과 신현태 교수 | 임상 설계 자문 | Track B |
| IP·사업화 | 특허법인 BLT | IP 전략, 사업화 컨설팅 | 공통 |
| 전략적 파트너 | 대웅제약 | 공동연구·기술이전 | Track B |

> ⚠️ BLT 역할: "IP·사업화" (절대 "IP·투자" 아님)

## 8. 플랫폼 투트랙 전략 사용 시 주의사항

### ⚠️ 서술 톤

| 항목 | 지침 |
|------|------|
| 플랫폼 강조 | 두 적응증을 관통하는 **경피 전달 펩타이드 플랫폼**의 범용성 강조 |
| Track A (피부암) | "동물 모델에서 치료 효능 확인" 수준. 과대 표현 지양 |
| Track B (탈모) | 정량적 차별성(40배/4.6배) 적극 활용, 주력 사업화 대상 |
| 화장품 | 독립 트랙이 아닌 "초기 매출 수단"으로 하위 배치 |
| 법인 설립 | 미설립 — "창업 예정" 톤 유지 (연구소기업 설립 계획) |

### ⚠️ 다른 전략과의 차이

| 비교 | 본 전략 (플랫폼 투트랙) | 투트랙 (화장품+치료제) | 치료제 단일 |
|------|----------------------|---------------------|-----------|
| 핵심 축 | 피부암 + 탈모 (2 적응증) | 화장품 + 치료제 (2 제품군) | 탈모 치료제 집중 |
| 화장품 위치 | 보조 (초기 매출 수단) | 주력 트랙 (Track A) | 거의 언급 안 함 |
| 강조 포인트 | 플랫폼 범용성 | 리스크 분산 | 임상 실현 가능성 |
| 적합 사업 | 창업선도대학, 예비창업패키지 | 일반 창업 지원 | 딥사이언스, TIPS |

### ⚠️ 금지 표현

| 금지 표현 | 대체 표현 |
|----------|----------|
| "전임상 검증 완료" (피부암) | "동물 모델에서 치료 효능 확인" |
| "저자극" (자사 기술) | "전신 부작용 없는 기전 기반" |
| "외부 투자 유치 계획" | "단계별 자금 조달 계획" |
| "특허법인 BLT (IP·투자)" | "특허법인 BLT (IP·사업화)" |
| CEO 역할에 "투자" | "파트너십·사업개발" |
| "혁신적" (자체 작성 파트) | 정량적 근거로 대체 |

### ⚠️ 권장 표현

- "경피 전달 펩타이드 플랫폼"
- "다적응증 확장 가능한 플랫폼 기술"
- "동물 모델에서 치료 효능 확인"
- "비침습적 국소 투여 기반 치료 전략"
- "플랫폼 범용성 입증을 통한 사업화 가치 극대화"

## 9. 예산 배분 가이드 (요약표 기준)

| 항목 | 비율 | 내용 |
|------|------|------|
| 연구개발 | 60% | 전임상 효능·안전성 보강, 제형 최적화 |
| 인건비 | 20% | 연구인력, 사업개발 인력 |
| 사업화·특허·마케팅 | 20% | IP 보강, 화장품 시제품, 시장 검증 |
