# Layer 2 — 전략 프리셋: 치료제 단일 (국소 탈모 치료제 집중)

> **적용 시점**: 의약품 개발에 집중하는 전략 사용 시 (딥사이언스, TIPS 등 치료제 중심 지원사업)  
> **사용법**: Layer 1(코어 팩트시트) + 본 파일 + Layer 3(스타일 가이드) 조합으로 문서 작성

---

## 1. 전략 방향 요약

| 항목 | 내용 |
|------|------|
| 전략명 | 기전 기반 국소 탈모 치료제 개발 (Single Track) |
| 핵심 방향 | 처방 기반 국소 Finasteride 복합제 개발 → 전임상·임상을 통한 의약품 허가 |
| 포지셔닝 | "전신 부작용 최소화 + 기전 기반 국소 전달" 차세대 탈모 치료제 |
| 핵심 논리 | 유효성이 검증된 약물(Finasteride)의 안전성과 전달 방식을 혁신하는 접근 → 신규 타깃 발굴 대비 임상 실현 가능성과 사업화 효율성이 높은 전략 |

---

## 2. 제품 포지셔닝

| 항목 | 내용 |
|------|------|
| 제품 유형 | 처방 기반 국소 Finasteride 제제 (펩타이드 나노운반체 탑재) |
| 핵심 성분 | 이중기능 펩타이드 운반체 + Finasteride |
| 적응증 | 안드로겐성 탈모 (Androgenetic Alopecia, AGA) |
| 기전 | 펩타이드 운반체의 경피 전달 + 자체 항염 기능 → Finasteride의 DHT 억제와 시너지 |
| 투여 경로 | 국소 경피 투여 (비침습) |
| 규제 경로 | 전문의약품 (비임상 → 임상 1/2/3상 → 품목허가) |

**치료제 단일 전략의 핵심 논거:**
- 이미 검증된 약물(Finasteride)을 기반으로 안전성과 전달 방식을 혁신 → 신규 타깃 발굴 중심의 고위험 개발 대비 임상적 실현 가능성과 사업화 효율성이 높음
- 경구 대비 40배, 경쟁 국소 대비 4.6배 투여량 감소 → 전신 노출 최소화라는 명확한 차별성
- 운반체 자체 항염 기능(carrier-free) → 단순 전달 기술이 아닌 치료적 시너지 구조

## 3. 사업모델 구조

### 3-1. 수익모델

| 수익원 | 시기 | 설명 |
|--------|------|------|
| 기술이전/라이선스아웃 | 전임상 PoC 이후 | 제약사 대상 기술 라이선스, 마일스톤+로열티 |
| 공동개발 | 임상 진입 전후 | 대형 제약사와 공동개발, 개발비 분담 + 공동 IP |
| 직접 개발·상용화 | 허가 이후 | 처방 기반 국소 치료제 직접 판매 (장기 옵션) |
| 플랫폼 확장 라이선스 | 중장기 | 동일 플랫폼으로 다른 적응증/약물 조합 라이선스 |

### 3-2. 가치 창출 구조

```
전임상 PoC 확보 → 임상 1상 → 임상 2상 → 임상 3상/허가
     │                │              │
     ▼                ▼              ▼
  Seed 투자       Series A       기술이전/공동개발
  기술가치 입증    임상 데이터     상용화 파트너십
```

## 4. 개발 로드맵 (이노베어 TIPS 3개년 기준)

> **사업 구조**: 1차년도(~12개월) / 2차년도(12~24개월) / 3차년도(24~36개월)
> **최종 목표**: 3차년도 말 전임상 데이터 패키지 완료 → IND 준비 단계 진입
> **전제**: 동물모델 효능 데이터(Finasteride 경피전달 탈모치료) 이미 확보 상태
> **⚠️ 사업 시작(M1): 2026년 6~7월 예정** (이노베어 선정 후) — 법인 없이도 R&D 기획·시장분석·제형 설계·IP전략은 M1부터 착수
> **⚠️ 법인 설립: 2026년 9월 예정 (M3경)** — 이후 대외 계약·투자유치·CRO 계약 등 법인 필수 활동 개시

## 7. 치료제 단일 전략 사용 시 주의사항

### ⚠️ 서술 톤

| 항목 | 지침 |
|------|------|
| 화장품/코스메슈티컬 언급 | **최소화**. "비의약 두피케어 제품 연계 가능성" 수준으로만 언급 |
| D2C/온라인 판매 언급 | **사용하지 않음**. 처방 기반 채널만 서술 |
| 기술 설명 중점 | 기전 기반 차별성, 정량적 우위(40배/4.6배), 임상 연계 가능성 |
| 사업모델 중점 | 기술이전/라이선스, 공동개발 중심 (제품 직접 판매보다 파이프라인 가치) |

### ⚠️ 금지 표현

- ~~"기능성 두피 제품을 MVP로 출시"~~ → 치료제 전략에서는 부적합
- ~~"D2C 기반 빠른 검증"~~ → 의약품은 임상으로 검증
- ~~"코스메슈티컬", "기능성 화장품"~~ → 치료제 집중 시 혼란 야기
- ~~"리텐션", "재구매율"~~ → 소비재 지표, 의약품에 부적합
- ~~"외부 투자 유치 계획"~~ → "단계별 자금 조달 계획"으로 대체
- ~~"저자극"~~ → "전신 부작용 없는 기전 기반"으로 대체

- ~~CEO 역할에 "투자" 포함~~ → "파트너십·사업개발"로 대체

## 8. 확장 가능성 (보조 서술용)

치료제 단일 전략에서도 플랫폼 확장성은 언급 가능하되, **보조적으로만** 서술:

| 확장 영역 | 서술 수준 |
|----------|---------|
| 탈모 외 적응증 (피부암, 아토피 등) | "플랫폼 기술의 적응증 확장 가능성" 정도로 1~2문장 |
| 신규 기전 (DKK1 억제 펩타이드) | "파이프라인 다변화 계획" 수준 |
| 두피케어 제품 | 언급하지 않거나, "화장품 라인 확보 시 CMC 학습 효과" 수준의 보조 언급만 |
