### 파일명 규칙

```bash
# KST 시간 자동 확인

- 형식: `YYYYMMDD_HHMM_클로드.docx`
- **반드시 `date -d '+9 hours'` 명령어로 KST 확인 후 입력** (UTC 임의 계산 금지)
- 저장 위치: `03 결과물/` (하위 폴더 생성 금지)

### XML 스타일 체계

| style val | 용도 |
|-----------|------|
| `11` | h3 소제목 |
| `20` | ㅇ 항목 (상위 bullet) |
| `30` | - 항목 (하위 bullet) |

    return format(_counter[0], '08X')

```
