# Layer 2 — 전략 프리셋: 투트랙 (두피케어 화장품 + 탈모 치료제)

> **적용 시점**: 화장품으로 초기 시장 진입 + 치료제로 중장기 임상 확장하는 듀얼 트랙 전략 사용 시  
> **사용법**: Layer 1(코어 팩트시트) + 본 파일 + Layer 3(스타일 가이드) 조합으로 문서 작성

---

## 1. 전략 방향 요약

| 항목 | 내용 |
|------|------|
| 전략명 | 리스크 분산형 듀얼 트랙 (치료제–두피케어 플랫폼) |
| Track A | 두피케어 제품(기능성/코스메슈티컬) → 조기 매출 + 시장 검증 |
| Track B | 국소 탈모 치료제(의약품) → 중장기 임상 진입 + 고부가가치 |
| 핵심 논리 | 의약품 장기 개발주기(비임상→임상→허가) 동안 동일 핵심기술로 화장품 시장 검증 및 초기 매출 확보, 화장품에서 확보한 피드백·제조 경험을 의약품 CMC 단계로 환류 |

---

## 2. 제품 포지셔닝

### Track A: 두피케어 제품 (화장품 트랙)

| 항목 | 내용 |
|------|------|
| 제품 유형 | 탈모 증상 완화 기능성 두피 제품 (leave-on 앰플/토닉) |
| 핵심 성분 | 펩타이드 단독 (약물 미포함) |
| 포지셔닝 | "두피 장벽을 고려한 전달 효율 기반" 탈모 증상 완화 솔루션 |
| 규제 경로 | 기능성 화장품 또는 일반 화장품 |
| 주요 KPI | 두피 자극 최소화, 4주 이상 지속사용 유지율 ≥ 80% |
| 핵심 근거 | 펩타이드 단독으로 유의미한 모발 성장 촉진 효능 확인 (전임상 데이터) |

**표현 규제 주의사항:**
- "탈모 치료"로 오인될 수 있는 표현 배제
- 허용 범위 내 표현: 두피 환경 개선, 모발 굵기 개선, 탈락 완화 등

### Track B: 탈모 치료제 (의약품 트랙)

| 항목 | 내용 |
|------|------|
| 제품 유형 | 처방 기반 국소 Finasteride 복합제 |
| 핵심 성분 | 펩타이드 운반체 + Finasteride |
| 포지셔닝 | 기전 기반 국소 전달 + 전신 노출 최소화 |
| 규제 경로 | 전문의약품 (비임상→임상→허가) |
| 적응증 | 안드로겐성 탈모(AGA) |
| 핵심 근거 | 경구 대비 40배↓ 투여량, Minoxidil 동등 이상 효능 |

## 3. 사업모델 구조

### 3-1. 수익모델

| 수익원 | 트랙 | 시기 | 설명 |
|--------|------|------|------|
| 제품 매출 (D2C) | Track A | 초기 | 탈모/민감피부 라인업 온라인 직접 판매 |
| 전문 채널 확산 | Track A | 확산기 | 피부과/두피케어/에스테틱 채널 입점 |
| B2B 공동개발 | Track B | 중기 | 마일스톤 + 개발비 + 공동 IP |
| 기술이전/라이선스 | Track B | 중장기 | 플랫폼 적용 라이선스아웃 |
| 원료 공급 | 양 트랙 | 중기~ | 펩타이드 원료 B2B 공급 |

### 3-2. 투 트랙 시너지 구조

```
Track A (화장품)                    Track B (치료제)
━━━━━━━━━━━━━━━━━━━━             ━━━━━━━━━━━━━━━━━━━━
- 빠른 시장 진입 (D2C)             - 비임상/임상 병행
- 소비자 피드백 확보                - 전임상 PoC 데이터
- 제형·자극감·사용성 데이터         - 규제 대응 문서화

         │                                  │
         └──────── 환류 ────────────────────┘
         (CMC 학습효과: 공정·안정성·품질지표)
```

## 7. 투 트랙 전략 사용 시 주의사항

### ⚠️ 문서별 톤 조절

| 문서 유형 | 강조 포인트 |
|----------|-----------|
| 화장품 지원사업 (예: 예비창업패키지) | Track A 중심, Track B는 "확장 가능성" 수준으로 언급 |
| 치료제 지원사업 (예: 딥사이언스) | Track B 중심, Track A는 "리스크 분산 및 CMC 학습" 수준으로 언급 |
| 대기업 협력 제안 (예: 이노베어) | 양 트랙 균형, 협력사 관점의 DDS 공동개발 강조 |
| 투자 IR | 양 트랙 병행 강조, 단기 매출(A) + 장기 가치(B) 스토리 |

### ⚠️ 포지셔닝 일관성 체크

- Track A 서술 시: "탈모 치료"가 아닌 "탈모 증상 완화", "두피 환경 개선" 표현 사용
- Track B 서술 시: "화장품" 언급 최소화, "처방 기반 국소 치료제" 중심
- 양 트랙 병행 서술 시: "동일 핵심기술 기반 듀얼 트랙 플랫폼" 프레이밍
