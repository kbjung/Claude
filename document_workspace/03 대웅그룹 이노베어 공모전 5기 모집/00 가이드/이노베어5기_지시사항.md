## 3. 작성 원칙

### 전략 방향

- **치료제 중심**으로 기술하되, 화장품은 "가능성" 수준으로 짧게 언급만
- "투트랙", "듀얼 트랙" 표현 사용 금지
- 포지셔닝: "기전 기반 국소 전달 + 전신 노출 최소화" 차세대 탈모 치료제

### 기술 파트 (섹션 1, 2)

- 보건의료 문서의 기술 설명, 기전, 선행연구 결과, 경쟁기술 비교 내용을 **거의 그대로** 사용
- 문장 구조·표현을 임의로 수정하지 말고, 이노베어 양식에 맞는 분량 조절과 서식 적용만 수행
- 수치(40배 감소, 4.6배 감소, Minoxidil 동등 이상 등)는 Layer 1 팩트시트와 정확히 일치시킬 것

### 사업 파트 (섹션 3, 4, 5)

- Layer 2(치료제단일) 전략 방향에 맞춰 작성
- Layer 1의 시장 수치를 정확히 인용
- 로드맵은 Layer 2의 3개년 구조 사용 (M1=2026.06~07, 법인설립=2026.09)
- 사업계획(4-1)과 연구수행계획(4-2)은 제임스 초안 기반, 치료제 전략에 맞게 톤 조정

### 특허·논문

- **5개 특허** 기재 (Layer 1 > §5-2 이노베어 기재 특허 참조)
- **논문 5편** 기재 (Layer 1 > §6-1 + §6-2)

### 이미지

- 요약 페이지에 기술 장점을 드러내는 1장짜리 핵심 이미지 삽입 위치 표시 (이미지 별도 제공 예정)

## 4. 제이콥 지시사항 이력

### 2025.12.23 — 초기 방향 설정

- TIPS 트랙으로 진행
- 대웅 DDS 수요기술 중 "만성질환(피부, 점막) 신규 흡수촉진 기술" 카테고리 매칭
- 탈모 당장 넣고, 아토피·통증 관련 치료제 운반도 포함하는 방향

### 2026.01.03 — 분량·적응증 구체화

- 전체 분량: 컴팩트 10장 또는 충실 20장 → 최종 20장으로 결정
- 탈모 + 아토피 중심으로 사업 이야기 전개

### 2026.01.20 — 제안서 작성 지시

- Jay가 이노베어 제안서 작성 주도
- 셀바스찬 제안서 참고하여 깔끔하게 작성
- 공고·양식 꼼꼼히 확인 + 문의사항 체크

### 2026.01.30 — 전체 일정·역할

- 이노베어 마감에만 맞추지 말고 가능한 빠르게 진행
- 보건복지, 딥사이언스, 이노베어 모두 낼 수 있도록 준비

### 2026.02 — 최신 작성 원칙

1. **기술 관련 내용은 수정하지 말고 거의 그대로 넣을 것** (보건의료 문서 기준)
2. **전략 방향**: 치료제 중심, 화장품은 가능성으로 짧게 (투트랙 직접 언급 X)
3. **TRL 단계**: 딥사이언스와 유사하게 전임상까지 진행 예정
4. **법인 설립 시점**: 2026년 9월 예정 — 간트차트·일정 전반에 반영
5. **딥사이언스 문서 참고**: 사업화 파트(섹션 3~4) 작성 시 반드시 참고. 전임상은 **2차년도 내내** 진행하는 구조
6. **사업화 파트**: 맞고 현실성 있고 구체성 있는 내용으로 작성. AI 추상적 내용 금지
7. **기수행 협의 실적**(Bio FD&C, Dt&CRO, P&K, 에피바이오텍 등) 활용 가능
8. **검증되지 않은 주장**(저자극, 균일 도포 우위, 국소 치료 선호 증가 등) 근거 없으면 삭제
9. **경쟁사 한계**: 우리 기술과 연결되는 내용만 사용
