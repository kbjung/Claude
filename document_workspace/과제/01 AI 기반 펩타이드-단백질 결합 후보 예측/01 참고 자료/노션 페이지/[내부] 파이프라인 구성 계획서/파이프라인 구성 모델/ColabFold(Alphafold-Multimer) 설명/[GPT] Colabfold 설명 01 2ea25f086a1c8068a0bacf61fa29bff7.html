[GPT] Colabfold 설명 01

1) 기본 개념

  • AlphaFold-Multimer: 여러 체인이 함께 접히는(co-folding) 복합체 3D 구조를 서열(+MSA/템플릿)로 예측하는 딥러닝 모델.
  • ColabFold: AF/AF-Multimer를 쉽고 빠르게 돌리게 해주는 파이프라인(특히 MMseqs2 기반 MSA로 속도·접근성 ↑). 모델 정확도 자체는 대체로 기본 AF와 동급.

2) ColabFold(멀티머)가 실무에서 “더 잘 되는 듯 보이는” 이유

  • 빠르고 넓은 MSA: MMseqs2로 동형서열 탐색을 고속화 → 더 많은/적절한 상동서열이 모여 입력 품질(MSA)이 개선되는 경우가 있음.
  • 샘플링 편의성: 리사이클/드롭아웃/시드 반복 등을 간단히 여러 번 돌려 좋은 포즈를 건질 확률이 커짐.
  • 핵심 포인트: 모델이 바뀌었다기보다 입력(MSA)·샘플링이 좋아져 성과가 좋아 보이는 케이스가 늘어남.

3) “가장 안정한(에너지 최소) 구조”를 고르나?

  • 아니요. 물리 에너지 직접 최소화가 아니라, 학습된 네트워크가 서열·MSA·템플릿 신호로 일관된 구조/인터페이스를 산출하고
    • ipTM(인터페이스 정확도), pTM(전역), pLDDT(국소), PAE(상대 배치 불확실성) 등 신뢰도 지표로 랭킹합니다.

4) “특정 결합 위치”를 알려주나?

  • 자동 라벨링은 아님. 다만 예측된 복합체 좌표로 **맞닿는 잔기(인터페이스)**를 바로 확인 가능.
  • PAE가 낮고 pLDDT가 높은 인터페이스는 비교적 신뢰도 ↑. 필요 시 PLIP 등으로 결합부위 특징(수소결합·소수성 등) 자동 주석화 가능.

보충: MSA와 MMseqs2 (초간단)

  • MSA(Multiple Sequence Alignment): 표적과 다수 상동서열을 정렬해 **공진화 신호(서로 짝지어 변하는 잔기)**를 뽑는 입력.
    • 깊이·커버리지가 좋을수록 모델이 인터페이스 제약을 더 잘 학습/활용.
  • MMseqs2: 초고속 동형성 탐색/클러스터링 도구. 매우 큰 DB에서 빠르게 상동서열을 모아 MSA를 구성.
    • ColabFold가 수십 배 수준 가속을 얻는 핵심. 보통 **속도↑ vs. 민감도↓**의 트레이드오프를 잘 조절해 실무 효율을 크게 높임.

평가 모델로서 설명

  • 간략 설명
    • 단백질 복합체의 3D 구조를 예측하는 딥러닝 모델(AlphaFold-Multimer 기반).
    • MMseqs2를 이용한 고속 MSA 생성으로 예측 속도와 접근성을 크게 향상.
    • 기본 AlphaFold와 동등한 수준의 높은 예측 정확도를 보임.
    • 파이프라인 적합성
      • 펩타이드-단백질 복합체의 3D 구조를 생성하는 파이프라인의 핵심적인 첫 단계.
      • 예측된 구조 파일(.pdb)은 Vina, PLIP, PRODIGY 등 후속 평가 단계의 필수 입력 데이터로 작용.
      • 자체 신뢰도 지표(ipTM, pTM)로 가장 가능성 높은 구조를 자동 선별하여 제공.
    • 신뢰도
      • 신뢰도: 최상 (State-of-the-Art) 👑
      • 근거:
        • 2020년 Nature지에 발표 후, 구조생물학 분야를 혁신한 모델.
        • 단백질 구조 예측의 오랜 난제를 해결한 공로를 인정받음.
        • 현재 구조 예측이 필요한 모든 연구의 foundational tool(기반 도구)로 사용.
        • 수만 회 이상의 논문 인용 횟수가 신뢰도를 증명.
    • 내부 평가 지표 설명
      • pTM : 한 체인의 전역 접힘 정확도에 대한 예측
      • ipTM : 체인 간 인터페이스(결합면) 배치 정확도에 대한 예측 → 결합 모델 신뢰도와 직결
      • pLDDT : 각 잔기(좌표)의 국소적 신뢰도
      • PAE : 두 잔기 사이 상대적 배치 오차(Å) 예측
