Pafnucy 설명

Pafnucy(발음: “paph‑nu‑sy”)는 단백질–리간드 결합 친화도(binding affinity)를 예측하기 위해 개발된 3D 컨볼루션 신경망 기반 모델입니다.

🔬 주요 개요

  • 입력 구조화

    리간드 중심의 20 Å 입방체 내 원자들을 1 Å 해상도 그리드로 표현하며, 각 격자는 19개의 원자 특성(화학적 유형, 전하, 수소결합 donor/acceptor 여부 등)으로 구성된 4D 텐서 형태입니다 (arXiv).

  • 네트워크 구조

    3개의 3D Conv 레이어(필터 수: 64, 128, 256)를 통해 공간 정보를 추출한 후, 3개의 Fully‑Connected 레이어(각각 1,000, 500, 200 뉴런; ReLU 활성화 및 dropout 적용)로 결합 친화도를 회귀 예측합니다 (PMC).

  • 데이터 & 학습 평가
    • 학습: PDBbind 2016 데이터셋(총 약 12,000개 복합체)
    • 검증: 1,000개 복합체 (refined set 중)
    • 시험: PDBbind Core Set (290개) 및 Astex Diverse Set (~73개)

      Classical scoring 함수들(예: Vina, MM/GBSA)을 능가하는 퍼포먼스를 입증했습니다 (PMC).

  • 해석 가능성 (Interpretability)
    • L2 regularization을 통해 필터 중요도를 분석할 수 있음
    • Voxel‑removal 실험을 통해 예측에 중요한 공간 영역(예: 수소결합, π–π 상호작용 등)을 시각적으로 확인 가능 (BioMed Central).

🧪 성능 지표

  • CASF‑2013 ‘scoring power’ 벤치마크 및 Astex 데이터셋에서 기존의 classical scoring function보다 더 낮은 오류(RMSE)와 높은 상관도를 기록함 (ResearchGate).
  • PDBbind v2016 Core Set 기준 RMSE 약 1.42 kcal/mol 수준으로 보고됨 (ACS Publications).

💡 활용 시나리오

  • 신약 후보 물질 선별(Virtual Screening): docking 후 구조 기반 스코어링 수행
  • 리드 최적화 (Lead Optimization): gradient 정보를 활용해 리간드 구조 개선 방향 제시
  • 구조 해석: voxel 제거 기반으로 중요한 상호작용 부위 시각화 및 분석 가능

🧭 요약 정리

항목내용
입력20 Å 격자, 각 격자 19개 원자 특성 포함
모델3D-CNN → Fully connected network
데이터셋PDBbind 2016 (train/val/test), Astex Diverse Set
성능classical scoring 함수 대비 우수한 예측력 (RMSE ≈ 1.4 kcal/mol)
해석 가능성L2 정규화 및 voxel-removal 통한 feature 중요도 분석

🧠 한계 및 주의사항

  • 회전 불변성 부재: 3D 그리드 방식은 데이터 증강(rotation augmentation)이 필요
  • 계산 비용: 그리드 생성 및 3D conv 처리 과정으로 메모리/연산량 소모가 큼
  • 최신 그래프 기반 GNN 접근법이 더 효율적이고 정확할 수 있음 (ResearchGate, BioMed Central, ACS Publications, PLOS, PMC).
