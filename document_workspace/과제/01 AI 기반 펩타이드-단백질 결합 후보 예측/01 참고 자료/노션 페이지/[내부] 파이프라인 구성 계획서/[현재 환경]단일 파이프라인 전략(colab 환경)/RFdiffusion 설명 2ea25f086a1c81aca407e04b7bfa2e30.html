RFdiffusion 설명

RFdiffusion은 단백질 구조 생성을 위한 딥러닝 기반의 확산 모델(diffusion model)입니다. 기존의 구조 예측 모델인 RoseTTAFold를 기반으로 하며, 단백질의 새로운 3차원 구조를 설계(생성)하는 데 초점을 둔 모델입니다.

🔍 핵심 개념 요약

항목설명
모델 이름RFdiffusion
기반 기술RoseTTAFold + Diffusion Models
주요 기능단백질 구조 생성 (protein backbone structure generation)
응용 분야단백질 디자인, 펩타이드 생성, 단백질-단백질 상호작용 설계 등
출판2022년 Baker Lab에서 발표

⚙️ 작동 방식

RFdiffusion은 denoising diffusion probabilistic models (DDPM)의 원리를 사용하여 무작위 잡음에서 점점 더 정확한 단백질의 3D 구조로 발전시킵니다.

기존 RoseTTAFold의 구조 예측 기능을 기반으로, "랜덤 구조 → 실제 단백질 구조" 로 점진적으로 복원해가며 새로운 구조를 생성합니다.

📌 특징

  • 백본 중심 설계: 아미노산 서열보다는 3차원 backbone 구조 자체를 생성
  • 유연한 조건 부여: 특정 접합 부위(anchor), 모양, 결합 정보 등을 제어하여 설계 가능
  • 다양한 생성 시나리오: 단백질-단백질 결합 구조 생성, 특정 모티프 포함 구조 생성 등

📂 활용 예시

  • 특정 단백질 타겟에 결합할 신규 펩타이드 설계
  • 항체-항원 상호작용 인터페이스 디자인
  • 인공 효소 활성 부위 생성
