# Layer 2 — 전략 프리셋: 치료제 메인 + 비의약 두피케어 보조

> **적용 시점**: 탈모 치료제를 메인 파이프라인으로, 비의약 두피케어를 초기 수익 보완용 보조 트랙으로 운용하는 전략 사용 시
> **사용법**: Layer 1(코어 팩트시트) + 본 파일 + Layer 3(스타일 가이드) 조합으로 문서 작성
> **1차 참고 문서**: 이노베어 사업계획서
> **최종 업데이트**: 2026.03.16

---

## 1. 전략 방향 요약

| 항목 | 내용 |
|------|------|
| 전략명 | 치료제 중심 사업화 전략 (비의약 보조 수익 병행) |
| 메인 파이프라인 | 탈모 치료제 (경피 전달, Finasteride 경구 대비 1/40) + 피부암 치료제 (펩타이드 약물 전달 확장) |
| 보조 파이프라인 | 비의약 두피케어 제품 (펩타이드 단독, 초기 매출 확보, 규제 진입 무관) |
| 핵심 논리 | 이중기능 펩타이드(피부투과 + 항염 치료 활성, 캐리어 프리) 기반의 경피 전달 기술로 탈모 치료제 시장의 미충족 수요를 해결하고, 동일 기술의 비의약 제품으로 초기 매출을 확보하는 구조 |

---

## 2. 제품 포지셔닝

### 메인: 탈모 치료제 (의약품 트랙)

| 항목 | 내용 |
|------|------|
| 제품 유형 | 처방 기반 국소 Finasteride 복합제 |
| 핵심 성분 | 이중기능 펩타이드 운반체 + Finasteride |
| 포지셔닝 | 기전 기반 국소 전달 + 전신 노출 최소화 |
| 규제 경로 | 전문의약품 (비임상, IND, 임상, 허가) |
| 적응증 | 안드로겐성 탈모(AGA) |
| 핵심 근거 | 경구 대비 1/40 투여량, 핀쥬베 대비 1/4.6, Minoxidil 동등 이상 효능 |
| 핵심 기술 | 이중기능 펩타이드 (피부투과 + 항염 치료 활성, 캐리어 프리) |

### 메인 확장: 피부암 치료제

| 항목 | 내용 |
|------|------|
| 대상 질환 | 흑색종 (피부암) |
| 핵심 근거 | 동물 모델에서 Imiquimod 국소 치료제 대비 우수한 치료능 확인 (Adv. Funct. Mater. 2025 게재) |
| 포지셔닝 | 펩타이드 약물 전달 플랫폼의 적응증 확장 |

### 보조: 비의약 두피케어 제품

| 항목 | 내용 |
|------|------|
| 제품 유형 | 두피케어 제품 (leave-on 앰플/토닉) |
| 핵심 성분 | 펩타이드 단독 (약물 미포함) |
| 포지셔닝 | 동일 펩타이드 기술 기반의 초기 수익 확보 제품 |
| 규제 경로 | 일반 화장품 |
| 역할 | 치료제 개발 기간 동안 초기 매출 확보 + 제형/제조 경험 축적 |
| 핵심 근거 | 펩타이드 단독으로 유의미한 모발 성장 촉진 효능 확인 (전임상 데이터) |

## 5. 시장 진입 전략

### 5-1. 메인 (치료제): 기술이전/공동개발 중심

| 단계 | 전략 |
|------|------|
| 1단계 | 전임상 PoC 데이터 패키지 완성 |
| 2단계 | 전략적 파트너(대웅제약 등)와 공동개발 협의 |
| 3단계 | 기술이전/License-out 또는 공동 임상 진입 |

### 5-2. 보조 (비의약 두피케어): D2C 중심

| 단계 | 채널 | 전략 |
|------|------|------|
| 1차 | D2C (자사몰/콘텐츠) | 전후 비교 콘텐츠, 전문가 리뷰 기반 |
| 2차 | 두피/피부과, 에스테틱 | 신뢰 기반 전문 채널 확장 |
| 3차 | H&B/오프라인 리테일 | 데이터 축적 후 단계 진입 |

### 5-3. 제조 전략

| 단계 | 전략 |
|------|------|
| 초기 | OEM/ODM으로 빠른 양산 및 품질 시스템 확보 |
| 확장 | 판매 증가 및 핵심 공정 검증 후 내재화/고도화 검토 |

- 메인(치료제) 서술 시: "처방 기반 국소 치료제" 중심, "화장품" 표현 최소화

- 전체 프레임: "이중기능 펩타이드 기반 경피 전달 플랫폼"으로 기술 통합
