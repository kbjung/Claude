## 1. PPT 레이아웃 프롬프트 (Claude / ChatGPT용)

```
PPT 1슬라이드에 BM 다이어그램을 만들어줘. 슬라이드 크기는 16:9(와이드).

### 파트 지칭 규칙
- A: 좌측 치료제 파이프라인 (Core Pipeline)
- B: 중앙 허브 (펩타이드 기반 경피 전달 플랫폼)
- C: 우측 두피케어 제품 파이프라인 (Supporting Pipeline)
- D: 수익 구조 (Revenue Model)
- E: 성장 전략 (Growth Strategy)
- F: 파이프라인 확장

### 전체 구조 (3열 + 하단 3행)

슬라이드를 크게 상단 영역(A·B·C)과 하단 영역(D·E·F)으로 나눈다.

**[상단 영역] A(좌) — B(중앙) — C(우) 3열 배치**

**B (중앙 허브)** — 슬라이드 상단 중앙에 배치
- "펩타이드 기반 경피 전달 플랫폼" 원형/육각형 허브
- 허브 안에 분자구조 아이콘 이미지(40×40px 정도) 삽입, 그 아래 텍스트
- B에서 좌측 A로 뻗어나가는 곡선 화살표 (틸 색상)
- B에서 우측 C로 뻗어나가는 곡선 화살표 (틸 색상)
- B에서 아래 D로 내려가는 화살표 (틸 색상) — 수익 구조와 연결

**A (좌측 — Core Pipeline: 탈모 치료제)** — B 왼쪽에 세로 배치
- "Core Pipeline: 탈모 치료제" 라벨 (레드/코랄 계열)
- 각 단계는 둥근 모서리 박스, 박스 왼쪽에 작은 아이콘(24×24px), 오른쪽에 텍스트
- 단계들을 세로 방향(위→아래)으로 배치, 각 단계 사이에 아래 방향 화살표:
  ① [🔬아이콘] 국소 탈모 치료제
       ↓
  ② [🧪아이콘] 전임상 검증
       ↓
  ③ [📋아이콘] 임상 시험 수행
- A 하단 캡션: "장기적 기업 가치 및 기술 경쟁력 확보"

**C (우측 — Supporting Pipeline: 비의약 두피케어 제품)** — B 오른쪽에 세로 배치
- "Supporting Pipeline: 비의약 두피케어 제품" 라벨 (딥블루 계열)
- 각 단계는 둥근 모서리 박스, 박스 왼쪽에 작은 아이콘(24×24px), 오른쪽에 텍스트
- 단계들을 세로 방향(위→아래)으로 배치, 각 단계 사이에 아래 방향 화살표:
  ① [🧴아이콘] 두피 케어 제품
       ↓
  ② [⚙️아이콘] OEM/ODM 생산
       ↓
  ③ [📈아이콘] 초기 매출 창출
- C 하단 캡션: "초기 시장 진입 및 현금 흐름 확보"

**[하단 영역] D → E → F 가로 행 배치 (위→아래 순서)**

**D (수익 구조 / Revenue Model)** — B 아래 가로 행
- "수익 구조" 라벨을 좌측에 세로 표기
- 4개 박스를 가로 균등 배치, 각 박스 상단에 작은 아이콘(24×24px), 아래에 텍스트:
  ① [💊아이콘] 치료제 상용화 | ② [🧴아이콘] 코스메슈티칼 제품 판매 | ③ [🤝아이콘] 기술 이전 및 파트너십 | ④ [💰아이콘] 전략적 투자

**E (성장 전략 / Growth Strategy)** — D 아래 가로 행
- "성장 전략" 라벨을 좌측에 세로 표기
- 4단계 화살표 타임라인(좌→우), 각 단계 번호 원 안에 표시:
  ①시드/Pre-A 투자 → ②제약/화장품 기업 전략적 파트너십 → ③글로벌 시장 진출 → ④IPO 또는 대규모 파트너십

**F (파이프라인 확장)** — E 아래 가로 행
- "파이프라인 확장" 라벨을 좌측에 세로 표기
- 3개 항목을 가로 화살표로 연결, 각 항목 왼쪽에 작은 아이콘(24×24px):
  [🔥아이콘] 염증성 두피 질환 → [💉아이콘] 피부 질환 치료제 → [🎯아이콘] 피부암 치료제(장기)

### 화살표 연결 정리
- B → A: 중앙 허브에서 좌측으로 뻗는 곡선 화살표
- B → C: 중앙 허브에서 우측으로 뻗는 곡선 화살표
- B → D: 중앙 허브에서 아래로 내려가는 화살표 (수익 구조와 연결)
- A 내부: ①→②→③ 세로 아래 방향 화살표
- C 내부: ①→②→③ 세로 아래 방향 화살표
- E 내부: ①→②→③→④ 가로 우측 방향 화살표
- F 내부: 항목 간 가로 우측 방향 화살표

### 디자인 가이드 (밝은 배경 스타일 — 프레젠테이션2_제이 참고)
- 배경: 흰색(#FFFFFF) 또는 매우 연한 그레이(#F5F7FA)
- 좌측 Core Pipeline 영역: 연한 민트/그린 배경(#E8F5E9 ~ #E0F2F1)으로 영역 구분
- 우측 Supporting Pipeline 영역: 연한 블루/시안 배경(#E0F7FA ~ #E3F2FD)으로 영역 구분
- 하단 성장전략/파이프라인 확장 영역: 연한 그레이(#ECEFF1) 가로 띠 배경
- 강조색: 진한 청록(teal) #00796B ~ #009688 — 화살표, 허브 테두리, 연결선에 사용
- 텍스트: 진한 다크그레이(#333333), 캡션은 미디엄 그레이(#777777)
- "Core Pipeline" 라벨: 레드/코랄(#E53935) 계열, "Supporting Pipeline" 라벨: 딥블루(#1565C0) 계열
- 중앙 허브: 흰색 원형에 틸 테두리, 안에 작은 아이콘 이미지 + 텍스트
- 수익구조 4개 박스: 흰색 카드 + 연한 그림자(drop shadow)
- 아이콘 이미지 삽입 규칙 (참고: 프레젠테이션2_제이.png 스타일):
  · A/C 각 단계 박스: 박스 왼쪽에 24×24px 아이콘, 오른쪽에 텍스트 (가로 나란히)
  · B 중앙 허브: 허브 중앙에 40×40px 아이콘, 아래 텍스트 (세로 정렬)
  · D 수익구조 박스: 박스 상단에 24×24px 아이콘, 아래 텍스트 (세로 정렬)
  · F 파이프라인 확장: 각 항목 왼쪽에 24×24px 아이콘
  · 아이콘은 텍스트를 보조하는 역할이므로 크기를 작게 유지할 것
- 폰트: 제목 24pt 볼드, 본문 14pt, 캡션 10pt
- 타이틀: 슬라이드 최상단에 "[제안서 별첨] 펩타이드 기반 경피 전달 플랫폼의 사업 모델 도식 (BM Diagram)"
```

## 3. 사용 가이드

| 순서 | 작업 | 도구 |
|------|------|------|
| 1 | 섹션1의 레이아웃 프롬프트로 PPT 뼈대 생성 | Claude / ChatGPT |
| 2 | 섹션2의 각 프롬프트로 아이콘 이미지 생성 (14개: B×1, A×3, C×3, D×4, F×3) | DALL-E / Midjourney / Ideogram |
| 3 | 생성된 이미지를 PPT placeholder에 삽입 | 수동 편집 |
| 4 | 텍스트(한글/영문 라벨) 직접 입력 및 정렬 | 수동 편집 |
| 5 | 색상·크기 미세 조정 후 최종 확인 | 수동 편집 |

> **Tips**: 이미지 생성 시 transparent background가 안 되는 도구는 `on a solid white (#FFFFFF) background`로 대체하면 PPT 밝은 배경과 자연스럽게 블렌딩됨.
