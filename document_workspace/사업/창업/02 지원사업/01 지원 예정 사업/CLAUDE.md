# 프로젝트 가이드 참조 규칙

## 1. 공통 가이드 (항상 참조)

`00 공통 가이드/` 폴더의 파일들을 항상 참조할 것:
- Layer1_코어팩트시트.md — 기술·데이터·IP 팩트
- Layer2_전략_[치료제단일 / 투트랙 / 플랫폼투트랙].md — 전략 방향·사업모델
- Layer3_작성규칙_스타일가이드.md — 문장 스타일·서식·검수 규칙

## 2. 사업별 가이드 (해당 사업 작업 시 추가 참조)

각 사업 폴더의 `00 가이드/` 내 파일을 함께 참조:

| 사업 | 폴더 | 사업별 가이드 파일 |
|------|------|-----------------|
| 딥사이언스 | `01 딥사이언스 창업 활성화 지원사업/00 가이드/` | (추가 예정) |
| 이노베어 5기 | `03 대웅그룹 이노베어 공모전 5기 모집/00 가이드/` | 이노베어5기_지시사항.md |
| 예비창업패키지 | `04 예비창업패키지/00 가이드/` | 예비창업패키지_지시사항.md, 예비창업패키지_양식매핑.md |
| 창업선도대학 | `05 실험실 특화형 창업선도대학 사업/00 가이드/` | 제이콥_지시사항.md, 혁신창업실험실_양식매핑.md |

## 3. 작업 순서

1. 공통 가이드 (Layer1~3) 읽기
2. 해당 사업의 `00 가이드/` 내 지시사항·양식매핑 읽기
3. 해당 사업의 원본 양식·참고자료 확인
4. 문서 작성
