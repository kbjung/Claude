# 프로젝트 가이드 참조 규칙

## 1. 공통 가이드 (항상 참조)

`00 공통 가이드/` 폴더의 파일들을 항상 참조할 것:
- Layer1_코어팩트시트.md — 기술·데이터·IP 팩트
- Layer2_전략_[치료제단일 / 투트랙 / 플랫폼투트랙].md — 전략 방향·사업모델
- Layer3_작성규칙_스타일가이드.md — 문장 스타일·서식·검수 규칙

`01 공통 참고 자료/` 폴더에 사업 간 공통으로 사용되는 참고 자료가 모여 있음:

- 05 예비창업패키지/ — 예비창업패키지 사업계획서 최종본 (0323)

## 3. 작업 순서

1. 공통 가이드 (Layer1~3) 읽기
2. 공통 참고 자료 확인
3. 해당 사업의 `00 가이드/` 내 지시사항·양식매핑 읽기
4. 해당 사업의 원본 양식·고유 참고자료 확인
5. 문서 작성
6. **자가검증** — 작성 완료 후 `00 공통 가이드/Layer3_작성규칙_스타일가이드.md`를 다시 읽고, 작성된 문서가 아래 항목을 준수하는지 점검·수정할 것:
   - 종결 어미 (명사형 종결 우선, 용언형 잔존 여부)
   - 계층 구조 (ㅇ/- 시스템 준수 여부)
   - Layer1 수치·명칭 정확성 (임의 변경 없는지)
   - Layer2 전략 방향과 일관성 유지 여부
   - 원본 양식의 목차·섹션 순서 준수 여부
   - 안내 문구(파란색·빨간색) 삭제 완료 여부
