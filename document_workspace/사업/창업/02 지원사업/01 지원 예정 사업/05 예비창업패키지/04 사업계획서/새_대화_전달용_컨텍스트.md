| 수정 항목 | 기존 값 | 변경 값 |

#### (B) Layer2_전략_투트랙.md — 전략 구조 전면 개편

| 기존 구조 | 변경 방향 |
|----------|----------|
| Track A = 화장품 (메인) | **삭제** → 치료제가 메인으로 전면 대체 |
| Track B = 치료제 (확장) | **메인으로 승격** — 탈모 치료제 + 피부암 치료제 |

- **이노베어 3장(사업화 전략) 내용을 거의 그대로 반영**

## 7. 셀바스찬 스타일 가이드 (이전 세션에서 확정)

1. **표 중심 레이아웃**: 핵심 내용은 반드시 표로 정리. 3문장 이상 산문 연속 시 표 전환 검토
2. **표 내 ○ 기호**: 비교표에서 해당 항목 표시 시 ○ 사용
3. **이미지 매 섹션**: 최소 1개 이미지/도식. 위치는 `【이미지 슬롯: 설명】`으로 표기
4. **캡션 표기**: 이미지·표 아래 (A), (B) 등 라벨
5. **회색 강조 박스**: ❖ 기호 + 회색 배경 → 핵심 특징·차별점 강조
6. **볼드+밑줄 강조**: 핵심 키워드에 **볼드**+밑줄 조합
7. **여백 최소화**: 빈 공간 없이 정보 밀도 극대화 (10p 한도)
8. **이미지 슬롯 표기**: `【이미지 슬롯: 펩타이드 피부 투과 모식도】` 형태
