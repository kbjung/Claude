- 각 페이지: **간트차트** (상단) + **상세 테이블** (하단)

## 슬라이드 2: 비임상 (Preclinical) 파이프라인

### 간트차트 (~3년, 균등 년차 간격)

```
             0          Y1          Y2          Y3
             ├───────────┼───────────┼───────────┤        소요기간 (시작~종료 시점)
① 분석법 개발 ████████████                          ~1년 (0→1년차)
② 효력 시험       ██████                            2~8주 (0.5→1년차)
③ 기전 연구       ██████                            효력시험 병행 (0.5→1년차)
④ GLP 독성             ████████████████              1~1.5년 (1→2.5년차)
⑤ PK 연구              ████████████████              독성시험 병행 (1→2.5년차)
⑥ CMC             ████████████████████████          ~2년 (0.5→2.5년차)
⑦ IND 준비                             ██████       3~6개월 (2.5→3년차)
```

### 단계별 상세

| 단계 | 주요 내용 | 기간 | 예상 비용 | 협력 업체 |
|------|----------|------|----------|----------|
| **① 분석법 개발** | HPLC/Mass 분석법 + 생체시료 분석법 확립 | 6개월~1년 | 별도 협의 (약 2~4억) | Dt&CRO |
| **② 효력 시험** | 양모 모델 + 남성형 탈모 질환 모델(DHT) / 미녹시딜 양성대조 | 2~8주 | 1,500~3,000만 | 에피바이오텍 |
| **③ 기전 연구** | 바이오마커 qPCR, 조직 분석 / 모낭 직경/밀도/무게 측정 | 효력시험 병행 | 효력+기전 / ~3,000만 | 에피바이오텍 |
| **④ GLP 독성** | 단회투여 → DRF(2주, 용량 범위 탐색) → 4주 반복투여 / 13주 반복투여 + 유전독성 3종 시험 | 1~1.5년 | 8억~12억 | Dt&CRO |
| **⑤ PK 연구** | ADME, 경피 특성 / 투여 부위 조직 분석 | 독성시험 병행 | 독성 패키지 포함 | Dt&CRO |
| **⑥ CMC** | 원료의약품 규격 설정(CoA), 안정성 시험(ICH 가이드라인), 제형 스케일업 | ~2년 | 약 3~5억 | 미정 |
| **⑦ IND 준비** | 비임상 데이터 통합 + 임상시험계획서 / 식약처 사전상담/제출 | 3~6개월 | RA/CRO 비용 (약 1.5~3억) | 미정 |

## 슬라이드 3: 임상 (Clinical) 파이프라인

### 간트차트 (~9년, 균등 년차 간격)

```
             0    1yr   2yr   3yr   4yr   5yr   6yr   7yr   8yr   9yr
             ├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤    소요기간 (시작~종료 시점)
Phase 1      ████████                                                  ~1.5년 (0→1.5년차)
Phase 2               ██████████████                                   ~2.5년 (1.5→4년차)
Phase 3                              ██████████████████                ~3년 (4→7년차)
NDA 허가                                                ████████████   ~2년 (7→9년차)
```

바 위 텍스트:
- Phase 1: 안전성+PK / 20~80명
- Phase 2: AGA 100~250명 / 용량탐색
- Phase 3: 대규모 300~3,000명
- NDA 허가: 품목허가 심사

### 단계별 상세

| 단계 | 주요 내용 | 기간 | 예상 비용(한국 단독) | 협력 업체/비고 |
|------|----------|------|----------|-------------|
| **Phase 1** | 안전성 + PK / 건강인·환자 20~80명 / 외용제: Phase 1/2 통합 가능 | 1~1.5년 | 10~30억 | OPIS |
| **Phase 2** | AGA 환자 유효성·안전성 / 100~250명 / 용량 탐색 | 2~2.5년 | 50~100억 원+ | OPIS |
| **Phase 3** | 대규모 확증 시험 / 300~3,000명 / 미녹시딜·경구 Fina 대조 | 2.5~3년 | 수백억 원 | OPIS |
| **NDA 허가** | 품목허가 신청 → 식약처/FDA 심사 | 1~2년 | RA 비용 | 미정 |

### 임상 협력 업체

| 업체 | 역할 | 미팅일 | 상태 | 핵심 메모 |
|------|------|--------|------|----------|
| OPIS | 글로벌 CRO | 2026.02 | 완료 | 30개국 네트워크, EU 임상 협력 |
| 대웅제약 | 전략적 파트너 | - | 지원 완료 (결과 대기) | 이노베어 5기 |
| CMC 전문업체 | 임상시험용 의약품(IMP) 제조, GMP | - | 미탐색 | 비임상→임상 공통 필요 |
| RA 전문가 | IND 신청, 임상 규제 전략 | - | 미탐색 | 비임상→임상 공통 필요 |

> **참고**: P&K 피부임상연구센터는 화장품 인체적용시험 기관(기능성 화장품 인증, 피부흡수도 평가)으로, 치료제 임상이 아닌 **화장품 트랙**에 해당.
